# IOU · 两个框重叠多少(交集 ÷ 并集)

## 一句话
衡量两个框有多重合的一个 0~1 数:它们的**交集面积 ÷ 并集面积**。完全重合=1,完全不沾=0。检测里用它判"预测框框得准不准",也用它给重叠框去重。

## 直觉 · 为什么不用"中心离多远"
判两个框像不像,光看中心距离不够——中心近但一个大一个小、或长宽比差很多,也不算框得好。IOU 一个数同时管住**位置 + 大小 + 长宽比**:只有当两个框**又对齐又一样大**,交集才接近并集,IOU 才高。所以它成了检测界通用的"框准度"标尺。

## 怎么做的 · 交集 ÷ 并集
```
交集 = 两个框重叠那块的面积
并集 = 两个框覆盖的总面积 = 面积A + 面积B − 交集
IOU = 交集 / 并集        （∈ [0,1]）
```
用处不止两处:① 训练时,[[yolo]] 的框置信度目标 = Pr(有物体) × IOU(预测框, 真值框),让网络学"框得越准置信度越高";② 推理去重时,[[non-max-suppression]] 把和高分框 IOU 超阈值(如 0.5)的其它框删掉;③ [[yolov2-yolo9000]] 的 [[dimension-clustering]] 用 `1-IOU` 当框形状距离,避免普通欧氏距离偏向小框。

## 数字例子 · 手算一次
两个 2×2 的框,一个在 [0,2]×[0,2],一个在 [1,3]×[1,3](右上偏移 1):
```
交集 = 重叠区 [1,2]×[1,2] = 1×1 = 1
面积A = 2×2 = 4,  面积B = 2×2 = 4
并集 = 4 + 4 − 1 = 7
IOU = 1 / 7 ≈ 0.14        → 重叠很少,框得不准
```
再看框得准的:两个框都在 [0,2]×[0,2] 但一个偏移 0.2 → 交集 ≈ 1.8×1.8=3.24,并集 ≈ 4+4−3.24=4.76,IOU ≈ 0.68。✓ 偏移越小 IOU 越接近 1;这正是训练想推高、去重想比较的那个数。
