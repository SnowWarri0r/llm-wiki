# Rubric-Based Evaluation · 拆细则打分

## 一句话
别让人/模型直接给"好不好"打一个总分，把它拆成一堆能各自逐项判定的小问题，每个独立打分再聚合。

## 直觉
"这张图真不真实?"——直接打 1-10 分,主观、难复现:你给 7 我给 5,谁对?换两个人甚至同一个人隔天打,分都不一样。

拆细则(rubric)的办法:把这个模糊判断**拆成一串具体、能验证的小问题**,每个只判 0/1/2(不满足/部分/满足):
- 光影物理合理吗?
- 材质质感真实吗?
- 有没有违反常识的穿帮?

每个小问题答案明确得多,两个人判同一条的一致性远高于"真实度几分"。把这些小分加起来,就得到一个**可复现、可拆解**的总分——还能告诉你模型**具体差在哪一条**,而不是只甩一个笼统的"7 分"。

它替代的是"整体打一个分"(holistic score)。整体分快但黑箱、噪声大;拆细则慢但稳、可解释。[[qwen-image-bench]] 把"图好不好"拆成 5 支柱 → 23 子能力 → **56 条细则**,正是这个思路推到极致。

## 怎么做的
```
1. 把目标能力拆成一棵树: 大维度 → 子能力 → 一条条可判定的细则
2. 每条细则定一个离散评分标准, 如 0=不满足 / 1=部分 / 2=满足 / N-A=不适用
3. 逐条打分(人或校准过的判官)
4. 聚合: 细则分 → 子能力 → 大维度 → 总分
```
关键:细则要"可验证"——问句具体到不同人看同一张图答案能对上,否则等于没拆。

## 数字例子
评一张图的「真实世界保真」这根支柱,它下面挂 3 条细则,每条 0/1/2:

```
细则①  光影物理合理?      → 2 (完全合理)
细则②  材质质感真实?      → 1 (塑料感, 部分)
细则③  有无常识穿帮?      → 2 (无穿帮)
─────────────────────────────────────
这根支柱得分 = (2+1+2) / (2×3 满分) = 5/6 ≈ 0.83
```

再假设另一根支柱「创意」3 条细则拿了 `[2,2,0] = 4/6 ≈ 0.67`。两根支柱等权聚合:
```
总分 = (0.83 + 0.67) / 2 = 0.75
```

✓ 自检:换个标注员来,他对"有没有穿帮"这种具体问题大概率也判 2(明确);但若让他直接给"这图整体几分",可能给 6 也可能给 8。**拆细则把分歧从"整体感觉"压到"逐条事实",一致性和可复现性就上来了**——而且总分 0.75 能一路回溯到"创意①②满分、③挂了",定位到具体短板。

## 跟整体打分的对比
| | 整体打一个分 | 拆细则 |
|---|---|---|
| 速度 | 快 | 慢(条目多) |
| 复现性 | 差(主观、噪声大) | 好(逐条明确) |
| 可解释 | 黑箱"7 分" | 能定位差在哪条 |
| 适合 | 粗筛 | 严肃 benchmark / 专业评测 |
