# MSE 最优 ≠ 内积无偏 · 压得最像,点积却系统性偏

## 一句话
按"重建误差最小(MSE)"设计的量化器,把向量压得最像原样;但拿压完的向量去算内积(点积),结果会**系统性地偏**(通常偏小)——因为量化误差跟向量本身相关。两个目标不是一回事,得分开治。

## 直觉 · "最像"和"点积准"是两码事
量化有两种"好":
- **MSE 好**:`x̂` 离 `x` 近(重建像)。这是逐点还原的目标。
- **内积无偏**:用 `⟨x̂, q⟩` 估 `⟨x, q⟩` 时,平均意义上不偏(检索、attention 打分要的是这个)。

问题:**MSE 最优的量化器,内积估计往往有偏**。直觉上,最优 MSE 量化器倾向把值往格子中心收(收缩),这个收缩是**朝着向量自己的结构**来的,于是 `⟨x̂,q⟩` 会**系统性偏小**——不是随机噪声,是每次都往同一方向偏,加再多样本也纠不回来。检索/attention 靠内积排序,系统偏差会歪掉排名。

## 怎么做的 · 两阶段:MSE 压主体 + QJL 残差去偏
[[turboquant]] 的解法是把两个目标拆开、各管一段:
```
阶段1  x̂ = MSE 最优量化器(x)      # 压主体,重建像,但 ⟨x̂,q⟩ 偏
阶段2  对残差 r = x − x̂ 做 1-bit QJL # 符号随机投影,给残差一个【无偏】内积修正
估内积 ⟨x,q⟩ ≈ ⟨x̂,q⟩ + QJL无偏修正(r,q)   # 主体低 MSE + 修正去偏 = 低 MSE 且内积无偏
```
[[quantized-jl]] 那一步是无偏的,专门补上阶段1 的系统偏差。

## 数字例子 · 有偏 vs 无偏,差在能不能纠回
真内积 `⟨x,q⟩ = 1.00`。
```
只用 MSE 量化器(有偏):
  ⟨x̂,q⟩ 每次都 ≈ 0.90   → 平均还是 0.90,系统性偏小 0.10,样本再多也停在 0.90 ✗
加 QJL 残差修正(无偏):
  ⟨x̂,q⟩ + 修正 = 0.90 + {单次乱跳但期望 0.10} → 多次平均 → 1.00 ✓
```
✓ 关键区别:偏差(bias)是**加样本纠不掉**的系统性错;方差(variance)是**加样本能平均掉**的随机抖。MSE 量化器给的是前者(致命),QJL 残差把它换成后者(可收敛)。所以最终估计既借 MSE 的低失真、又拿回内积的无偏。
