# TDM · 轨迹分布匹配（沿整条去噪路都对分布，少步还稳）

## 一句话
把 [[dmd-distillation]]（DMD）的"只匹配终点出图分布"推广到"沿去噪轨迹的多个点都匹配分布"，于是少步生成更稳、步数能灵活选、超参还少。

## 直觉 · 从"只看终点"到"看整条路"
[[dmd-distillation]]（DMD）蒸馏的目标是：学生（少步）画出来的**最终分布**跟老师（多步）的一样。这对**一步/极少步**很好用，但只盯终点，中间怎么走没人管——想做"灵活多步"（有时 2 步、有时 8 步）时不好控。

**TDM（Trajectory Distribution Matching，轨迹分布匹配）**的改法：不只在终点对分布，而是在**去噪轨迹上的好几个时间点**都要求"学生此刻的分布 ≈ 老师此刻的分布"。把约束从一个点摊到整条路上，少步生成的每一跳都被钉在老师的轨迹分布上 → 更稳、可以灵活选步数。[[krea-2]] 比过 DMD / DMD2 / Decoupled DMD / piFlow / APT，最后选 TDM，理由就是**好调、超参少**。

## 怎么做的 · 轨迹是什么、匹配什么
[[flow-matching]] 把生成看成从纯噪声沿一条路积分到数据。老师用很多小步（小心地走），学生想用几大步跳完：

```
时间 t:   1.0(纯噪声) ──────────────────────────► 0(图)
老师:     · · · · · · · · · · · · · · · ·   ~28 小步，每步挪一点
学生:     ●─────────●─────────●─────────●     4 大步，每步一大跳
            t=1.0    t=0.7    t=0.4    t=0.1   ← 在这些"路标"上对分布
```
- **DMD**：只要求最右边那个 ●（终点）的分布对上老师。
- **TDM**：要求每个 ● 这一刻，学生群体的分布都 ≈ 老师走到同一 t 时的分布。

## 数字例子 · 一条 1D 玩具轨迹
把一张图简化成 1D 的一个值，从噪声 `x=1.0` 走到数据 `x=0`。老师走 4 小步、学生想 2 大步跳完，路标设在 `t=0.5`（中点）和 `t=0`（终点）。

老师（每步 ×0.6，模拟"小心地走"）：
```
1.00 → 0.60 → 0.36 → 0.22 → 0.13   （4 步，最后落在 ~0.13 附近）
            ↑ t=0.5 这一刻老师在 0.36 附近
```
学生只两跳，TDM 要求它在两个路标都对上老师那一刻的分布：
```
路标 t=0.5： 学生第一跳要落在【0.36 附近的分布】← 跟老师中途对齐
路标 t=0  ： 学生第二跳要落在【0.13 附近的分布】← 跟老师终点对齐
```
对比只用 DMD：只约束终点 0.13，中间跳到哪没人管——学生可能第一跳直接冲过头（比如跳到 0.05），再硬拐回来，少步时容易抖。TDM 在 0.36 这个**中途路标**也钉住，等于给少步路径加了"中途检查站"。
```
✓ 自检：约束点数 1(DMD) → 多个(TDM)。多一个中途路标，学生的少步轨迹
  就少一分乱窜的自由 → 2/4/8 步都能稳出，且不用为每个步数重调超参。
```
> 类比：DMD 只考"终点站到没到对的地方"；TDM 沿途还设了几个打卡点，每个都要对——少走几站也不会跑偏。

## K2 Raw vs K2 Turbo
[[krea-2]] 用 TDM 做时间步蒸馏，发两套权重：
- **K2 Raw**：未蒸馏的基座，留给做微调 / 后训练研究的人。
- **K2 Turbo**：**引导蒸馏 + 时间步蒸馏**双管齐下（[[guidance-distillation]] 去掉 CFG 的双次前向、TDM 把步数压到几步），少步快出图。
