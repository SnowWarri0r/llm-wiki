# Inductive Bias · 归纳偏置

## 一句话
模型架构里**预先写死的"对世界的假设"** —— 你越早写得对, 数据少时越省; 数据多时反而是包袱。

## 直觉 · 老员工 vs 新员工

公司招两个程序员:

- **老员工 A**: 干了 20 年这个业务, 上来就知道"这种问题用 Redis、这种用 Kafka"。培训成本低, 5 个项目就能上手
- **新员工 B**: 经验为零, 但学习能力强。要看完 100 个项目才能形成自己的判断, 但一旦形成判断, **可能比 A 还准**, 因为 A 的经验里有过时的部分

**这就是归纳偏置的本质**。模型架构 = 老员工的"经验", 是设计者预先塞进去的对问题的判断。数据 = 项目。

**少数据时, 强偏置赢**（老员工省培训成本）; 大数据时, 弱偏置赢（新员工不被过时经验拖累）。

## CNN 的归纳偏置

ResNet / VGG 这些 CNN 架构里, **三条假设被硬编码进结构**:

1. **局部性 (locality)**: 一个像素只跟它周围的像素有关。卷积核 3×3, 一次只看 9 个像素
2. **平移不变性 (translation equivariance)**: 同一个卷积核在图像所有位置共享。猫在左上角和在右下角, 卷积核响应模式一样
3. **层次结构 (hierarchy)**: 通过 pooling 逐层降采样, 浅层看局部细节, 深层看全局语义

**这三条对自然图像几乎都成立**, 所以 CNN 在 ImageNet 时代横扫 —— 130 万张图就够它学好。

## ViT 的"无知"

ViT 把 CNN 的三条假设全扔了:

| 假设 | CNN | ViT |
|---|---|---|
| 局部性 | 卷积核 3×3 强制 | 没有, attention 跨全局 |
| 平移不变性 | 卷积权重共享 | 没有, 每个 patch 位置独立 |
| 层次结构 | pooling 逐层降采样 | 没有, 所有 layer 同分辨率 |

**ViT 唯一保留的偏置: 把图切成 patch 这一步** —— 假设 16×16 邻域里的像素相关, 算一个 patch 内部的事。除此之外, **patch 之间的所有关系都靠 attention 从数据里学出来**。

## 数据 vs 偏置的 trade-off

ViT 论文里这张图(类似):

```
准确率
  ↑
  │              CNN
  │       ╱ ─ ─ ─ ─ ─ ─    ViT
  │    ╱             ╱
  │  ╱            ╱
  │ ╱          ╱
  │           ╱     交叉点 ≈ ImageNet-21k 规模
  │        ╱
  │     ╱
  └──────────────────────→ 数据规模
   ImageNet  ImageNet  JFT
   -1k       -21k      -300M
```

**交叉点之前**: CNN 赢。它的"祖传家产"用对了, ViT 还没学到位

**交叉点之后**: ViT 赢, 而且越往后差距越大。CNN 的假设虽然对, 但**只能学到那种程度**; ViT 从无知开始, 但最终能学到更精细的关系（比如"这个 patch 跟图像另一端的某个 patch 有关"这种 CNN 卷不到的远程依赖）

## 这是个更一般的规律

| 领域 | 强偏置方案 | 弱偏置方案 | 大数据时谁赢 |
|---|---|---|---|
| 视觉 | CNN | ViT | ViT |
| NLP | RNN / LSTM (序列性) | Transformer (全局 attention) | Transformer |
| 语音 | spectrogram + CNN | Whisper (Transformer on mel) | Transformer |
| 物理 / 化学 | 等变神经网络 | 大 Transformer + 数据 | 还在打 |

这就是 Sutton 那篇 [[bitter-lesson]] 说的: **过去 70 年 AI 历史, 一次又一次证明 "通用方法 + 大算力 + 大数据" 打败 "把人类知识写进结构里"**。

## 工程上的实战含义

如果你训自己的模型:

- **数据少 (< 1M)**: 别瞎用 ViT, 老老实实 ResNet / EfficientNet
- **数据中等 (1M - 100M)**: 看任务, ViT 用足够强的 regularization (DeiT 那套) 才能赢
- **数据多 (> 100M)**: ViT / Swin 系列, 不要犹豫

或者用预训练好的 ViT (CLIP / DINOv2) → 微调到自己的小数据集上 —— **预训练阶段已经替你"看了"几亿张图, 弱偏置的代价已经摊销过了**。
