# 归一化流 · 用可逆变换把简单分布掰成复杂分布

## 一句话
拿一串**可逆**的变换,把一个简单分布(高斯)一步步掰成任意复杂的分布;因为可逆,既能正向采样、又能反向精确算出任一点的概率(靠变量替换 + 雅可比行列式)。

## 直觉 · 为什么要"可逆"
生成建模常要一个既能采样、又能算准概率的分布。简单高斯两样都行但太死板;复杂分布表达力够但概率难算。归一化流的折中:**从简单高斯出发,套一串可逆变换 f**。

- 正向:采个高斯噪声 ε,过 f 得到复杂样本 z=f(ε)。
- 反向:给个 z,用 f⁻¹ 退回 ε,再用"变量替换公式"精确算出 z 的概率。

可逆是关键——不可逆就退不回去、算不了概率。每层通常设计成好求逆、雅可比好算(如仿射耦合层)。

## 怎么做的 · 变量替换 + 雅可比
```
z = f(ε),  ε ~ 简单高斯
log p(z) = log p(ε)  −  log |det (∂f/∂ε)|
            └─简单可算─┘   └─这层"拉伸了多少"的修正─┘
堆很多层 f₁∘f₂∘…∘f_k → 简单高斯逐步变复杂;每层的 log|det| 累加
```
雅可比行列式 = 这个变换把空间"拉伸/压缩"了多少;概率要按拉伸量修正(拉开了密度就摊薄)。

## 数字例子 · 一维流的概率修正
最简单的可逆变换 z = f(ε) = 2ε + 1(拉伸 2 倍、平移 1):
```
ε ~ N(0,1)。取 ε=0.5 → z = 2·0.5 + 1 = 2.0
导数 dz/dε = 2 → |雅可比| = 2
p(z=2.0) = p(ε=0.5) / 2          # 拉伸 2 倍 → 密度摊薄成一半
         = 0.352 / 2 = 0.176
反向:给 z=2.0 → ε=(2.0−1)/2=0.5 精确退回 ✓
```
✓ 拉伸 2 倍把密度摊薄一半,这就是 log|det| 修正在做的事。真实归一化流把 2ε+1 换成很多层非线性可逆变换,就能把高斯掰成任意复杂形状,同时每一点概率都算得准。[[vits]] 就用它把"简单文本先验"掰复杂,去够音频后验。
