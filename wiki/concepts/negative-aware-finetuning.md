# 负样本感知微调 · NFT · 用奖励切正负,在前向过程上把 RL 折成监督损失

## 一句话
给每张生成图打个奖励 `r∈[0,1]`,按 `r` 把样本软切成正堆(高奖励)和负堆(低奖励);让一个模型同时学"正速度"和"负速度",正负之差自然指向"该往哪改"——于是扩散的在线 RL 退化成一个**加权的监督 [[flow-matching]] 损失**,不碰似然、不碰反向采样轨迹。

## 直觉 · 为什么要绕开反向、回到前向
扩散做 RL 卡在似然算不出来([[diffusion-rl-likelihood-barrier]])。主流绕法(FlowGRPO)是把**反向去噪**切成多步马尔可夫过程让概率可算,再套 [[grpo]]——但反向路有无数条,盯着它训会锁死采样器、还可能让模型偏离真正的前向过程。

NFT 的洞察一句话:**前向加噪只有一条、反向去噪有无数条**。既然前向是唯一且良定义的,那就把 RL 做在前向上。前向过程正是 [[flow-matching]] 的训练地盘——所以整件事能接回最朴素的监督训练,几乎不改代码。

## 怎么做的 · 奖励软切 + 单模型隐式装正负
```
1. 用任意 solver(ODE/SDE/高阶都行)采一批干净图 x₀,各打奖励 r∈[0,1]
2. 概念上切两堆:
     正分布 π⁺ ∝ r · π_old        (高奖励样本权重大)
     负分布 π⁻ ∝ (1−r) · π_old    (低奖励样本权重大)
3. 一个 velocity 网络 v_θ 隐式同时表示正负两副动态(围绕冻结的旧模型 v_old):
     v_θ⁺ = (1−β)·v_old + β·v_θ      (正/被引导的速度)
     v_θ⁻ = (1+β)·v_old − β·v_θ      (负速度)   ← 两者对称于 v_old
4. 监督损失(纯 flow matching,不是 policy gradient):
     L = E[ r·‖v_θ⁺ − v‖² + (1−r)·‖v_θ⁻ − v‖² ]
   高奖励样本(r→1)把 v_θ⁺ 拉向它的真速度、低奖励(r→0)把 v_θ⁻ 拉向它的真速度
```
正负只要存**干净图**就能算,不用整条采样轨迹;`r` vs `1−r` 的对比权重 + 隐式参数化,把强化信号揉进了普通的监督目标。

## 数字例子 · 隐式参数化对称在 v_old 周围
取标量简化(真实是逐维速度向量):旧模型速度 `v_old=1.0`、模型当前输出 `v_θ=1.4`、引导强度 `β=0.5`。
```
v_θ⁺ = (1−0.5)·1.0 + 0.5·1.4 = 0.5 + 0.7 = 1.2     (正速度)
v_θ⁻ = (1+0.5)·1.0 − 0.5·1.4 = 1.5 − 0.7 = 0.8     (负速度)

自检①:v_θ⁺ + v_θ⁻ = 1.2 + 0.8 = 2.0 = 2·v_old   → 正负永远对称夹着 v_old ✓
自检②:v_θ⁺ − v_θ⁻ = 0.4 = 2β·(v_θ − v_old) = 2·0.5·0.4 ✓  → 这个缝隙 = 模型离旧模型多远
```
✓ 含义:模型只动一个量 `v_θ`,正速度往上抬到 1.2、负速度对称压到 0.8。训练时高奖励样本把"上抬的那副"(v_θ⁺)拉准、低奖励把"压下的那副"(v_θ⁻)拉准,两股梯度合起来,就把 `v_θ` 推向"正比负更该走"的方向。采样时只用 `v_θ⁺`(被引导的正速度)——等于把 [[reinforcement-guidance]] 内建进了一个模型,不再需要 CFG 的两套模型。最优解 `v_θ* = v_old + (2/β)·Δ`,小小的改进方向 Δ 被 `2/β` 放大着学进去。

## 最优解从哪来 · 把损失对 v_θ 求导置零
`v_θ* = v_old + (2/β)·Δ` 不是凑的,是解最优。标量推一遍。

**先备:把 E[·] 当"加权平均",两条规则就够。** `E[X]` 读作"X 在这批样本上的平均值"。规则①常数(对每个样本都一样的量)能拎到平均号外:`E[c·X]=c·E[X]`;规则②加减拆得开:`E[X−Y]=E[X]−E[Y]`。<strong>谁是常数谁是变量是关键</strong>:`v_θ⁺/v_θ⁻/v_old` 是模型在某点 x_t 的输出,对每个样本都一样 = 常数能拎出;只有奖励 `r` 和目标速度 `v` 逐样本变。用两个样本(奖励取 0/1 最直观)把符号坐实:
```
样本 A: r=1, v=1.4    样本 B: r=0, v=0.6
ρ⁺=E[r]=0.5(好样本占比)   ρ⁻=E[1−r]=0.5
v_old=E[v]=(1.4+0.6)/2=1.0(所有样本平均)
v⁺=好样本里 v 的平均=1.4   v⁻=差样本里 v 的平均=0.6   （按奖励把样本分两桶:好→正桶、差→负桶）
```

**① 求导:两项梯度同向叠加(这就是那个 2)。** 单样本损失 `ℓ = r(v_θ⁺−v)² + (1−r)(v_θ⁻−v)²`,两副速度都由同一个 `v_θ` 来、β 符号相反:`dv_θ⁺/dv_θ=+β`、`dv_θ⁻/dv_θ=−β`。
```
dℓ/dv_θ = 2β·[ r(v_θ⁺−v) − (1−r)(v_θ⁻−v) ]
```
负项带 −β,梯度也把 v_θ 往同方向推 → 正样本"抬正速度"和负样本"压负速度"两股劲<strong>叠加</strong>,不抵消。这就是 2 的来源。

**② 置零 → 奖励加权残差平衡。** L 是个开口朝上的碗(平方和),<strong>碗底那点切线是平的=斜率为 0</strong>,所以"找最优 v_θ" = "找 dL/dv_θ=0 的点"。总斜率 = 各样本斜率的平均 `E[dℓ/dv_θ]=2β·E[r(v_θ⁺−v)−(1−r)(v_θ⁻−v)]`,令它=0、除掉 2β、拆减法移项 → `E[r(v_θ⁺−v)]=E[(1−r)(v_θ⁻−v)]`。左边再按"拎常数+拆减法"展开(v_θ⁺ 是常数;`E[rv]=ρ⁺v⁺` 因 r=0/1 时清掉差样本只剩好样本):
```
E[r(v_θ⁺−v)] = v_θ⁺·E[r] − E[rv] = ρ⁺·v_θ⁺ − ρ⁺·v⁺ = ρ⁺(v_θ⁺−v⁺)
右边同理 = ρ⁻(v_θ⁻−v⁻)，于是
   ρ⁺(v_θ⁺−v⁺) = ρ⁻(v_θ⁻−v⁻)   …(★)   （好样本拉准正速度的劲 = 差样本拉准负速度的劲）
```
把碗画出来看(两样本,β=0.5,v_old=1.0):`v_θ⁺=0.5+0.5v_θ`、`v_θ⁻=1.5−0.5v_θ`。A 只有正项、B 只有负项:
```
L = [ (v_θ⁺−1.4)² + (v_θ⁻−0.6)² ]/2 = [ (0.5v_θ−0.9)² + (0.9−0.5v_θ)² ]/2 = (0.5v_θ−0.9)²
标准抛物线,碗底在 0.5v_θ−0.9=0 → v_θ=1.8 = 最优解 ✓
查 (★) 平不平衡(左=(v_θ⁺−1.4)/2,右=(v_θ⁻−0.6)/2):
   v_θ=1.0: 左=−0.2 右=+0.2 不等 → 没到碗底
   v_θ=1.8: 左= 0   右= 0   相等 → 碗底,斜率=0 ✓
```

**③ 两个事实把它压扁。**
```
事实A  v_old = E[v] = ρ⁺v⁺ + ρ⁻v⁻        旧模型 = 正负两桶的加权平均(=班级均分=男均×男比+女均×女比)
          数字: 0.5·1.4 + 0.5·0.6 = 1.0 ✓
事实B  Δ = ρ⁺ρ⁻(v⁺−v⁻)                  改进方向(从负桶指向正桶 v⁺−v⁻, ρ⁺ρ⁻ 只是缩放)
          由事实A: v⁺−v_old = (1−ρ⁺)v⁺−ρ⁻v⁻ = ρ⁻(v⁺−v⁻)，故 ρ⁺(v⁺−v_old)=ρ⁺ρ⁻(v⁺−v⁻)=Δ
          也 = ρ⁻(v_old−v⁻)(论文 Thm 3.1)；数字: 0.5·0.5·0.8 = 0.2 ✓
```

**④ 代入 (★) 解 v_θ*。** 归拢 v_θ / v_old / v± 各项(用 ρ⁺+ρ⁻=1),中括号那坨用事实A 化简恰好 = 2Δ:
```
β·v_θ* = β·v_old + [ρ⁺v⁺ − ρ⁻v⁻ − (ρ⁺−ρ⁻)v_old] = β·v_old + 2ρ⁺ρ⁻(v⁺−v⁻) = β·v_old + 2Δ
⟹ v_θ* = v_old + (2/β)·Δ
```

`2/β` 的含义:**2** = 正负梯度同向叠加;**1/β** = 模型偏离 v_old 的部分只按 β 倍漏进被引导速度(`v_θ⁺=v_old+β(v_θ−v_old)`),所以要体现改进量 Δ,v_θ 得学一个放大 1/β 倍的偏移——β 越小,v_θ 偏得越狠。

### 数字核验
```
取 ρ⁺=0.5、v⁺=1.4、v⁻=0.6 → v_old=ρ⁺v⁺+ρ⁻v⁻=1.0，Δ=ρ⁺ρ⁻(v⁺−v⁻)=0.25×0.8=0.2
β=0.5：v_θ* = 1.0 + (2/0.5)×0.2 = 1.0 + 4×0.2 = 1.8
回代：v_θ⁺ = (1−0.5)·1.0 + 0.5·1.8 = 1.4 = v⁺ ✓（奖励对半时,最优被引导速度正好复现正分布速度）
```
