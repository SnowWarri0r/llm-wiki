# Style Reference · 只借参考图的"风格"，别把它的"内容"也搬来

## 一句话
用文字生成、再拿一/多张参考图导风格；核心难题是别让参考图的主体/物件钻进结果（内容泄漏），靠自监督训练 + 偏好优化把"风格"和"内容"两条通道拆开。

## 直觉 · "风格"和"内容"没有清晰边界
用户想要"我的猫，画成梵高那种笔触"。参考图（梵高的画）该贡献的是**笔触、色调、肌理**（风格），不该贡献的是**漩涡、村庄、星星**（内容）。可一张图里风格和内容是缠在一起的，模型很容易**连内容一起抄过来**——这就是 [[krea-2]] 说的最常见翻车：**内容泄漏**（content leakage），结果图里冒出参考图的主体。

更麻烦的是**数据稀缺**：图像编辑能从视频里挖"同一物体不同状态"的配对，但"同一内容套不同风格"的高保真配对**没法大量现成获取**。

## 怎么做的 · 自监督造对 + 偏好优化收尾
没有现成配对，就**自己造**（自监督）。[[krea-2]] 说用了"a novel self-supervised technique"，但**没公开具体构造**。这类问题的通用思路是制造 `(内容, 风格参考, 目标)` 三元组而不依赖人工配对，常见两种：
```
思路A 同图增广：把一张图做成"风格参考"(只留色调/笔触统计、打乱或抹掉内容)，
       要求模型在【另一个内容】上重建出这种风格 → 逼风格通道只学风格
思路B 自己当目标+风格源：同一张图既当目标又当风格来源，但把内容信息遮住，
       逼模型只能从"风格通道"取信息、从"内容通道(文字)"取主体
```
两者都在结构上**强制风格和内容走不同通道**，让模型学会"风格可借、内容归文字"。

**再接偏好优化**（见 [[direct-preference-optimization]] 思路）：生成一批结果，偏好"风格像、且内容没泄漏"的样本，往那个方向调——把"少泄漏"这个难量化目标，变成"两个里选泄漏更少的那个"的偏好信号。再加上**多风格平滑混合**、每个参考**强度连续可调**。

## 一个具体场景走一遍
```
参考图 = 梵高《星空》     用户 prompt = "一只猫"
想借: 漩涡状厚涂笔触、蓝黄对比色调
别借: 星空、漩涡云、村庄(这些是参考图的【内容】)

泄漏(坏): 出图里猫 + 莫名其妙的漩涡云/村庄 → 内容串味
目标(好): 你的猫，披着梵高式厚涂笔触和蓝黄调 → 内容是你的，风格是它的
强度旋钮: 0.3=淡淡梵高味；0.9=浓重梵高味
```
✓ 自检：判断成败就一条——**结果的"是什么"(内容)该由文字定，"长什么质感"(风格)才由参考图定**。泄漏 = 参考图擅自决定了"是什么"。

> 报告披露程度：Krea 只给了"自监督 + 偏好优化两段 + 压内容泄漏 + 多风格混合 + 强度可调"的**框架**；自监督怎么造对、PO 用什么奖励**都没细说**。上面思路 A/B 是这类任务的通用解法，不是 Krea 逐字实现。
