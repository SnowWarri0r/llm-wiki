# FFT · 快速傅里叶变换

## 一句话
傅里叶变换把一段信号拆成各频率的强度（像棱镜拆光）；FFT 用"偶/奇折半 + 单位根对称"的分治，把朴素的 N² 砍成 N·log N，让音频频谱、卷积加速、MP3/JPEG/MRI 全都跑得动。

## 它要解决的痛点
DFT 测每个频率的办法，是拿"转速 k 的转子"扫一遍全部 N 个采样（对齐就共振、对不上就抵消）。N 个频率 × N 个采样 = **N² 次乘加**。N=100 万时是 10¹² 次，没法实时。

## 核心贡献
- **分治拆偶/奇**：把 N 点按下标拆成偶半 + 奇半，整体 DFT 由两个 N/2 的半边 DFT 拼出，递归到底 → `T(N)=2T(N/2)+O(N)` = **N·log N**。
- **单位根对称**：转子 `w=e^(−2πi/N)` 转半圈反号 `w^(k+N/2)=−w^k` —— 让一次乘法 `w^k·O` 加减各取一次就喂**两个**输出，这是省下来的根源。
- **蝴蝶 + twiddle**：`X[k]=E+w^k·O`、`X[k+N/2]=E−w^k·O`，堆 log N 层蝴蝶 = 整个 FFT。twiddle = 中间乘的旋转因子。
- **实现细节**：radix-2 要求 N 是 2 的幂；递归到底的输入顺序是 bit-reversal（下标二进制位反转）；IFFT 同算法、twiddle 取共轭、除以 N。

## 关键概念 → 概念页
- [[convolution]] · 卷积定理：时域卷积 = 频域逐点乘，大核可走 FFT 加速
- [[log-mel-spectrogram]] · STFT = 把信号切窗、每段做一次 FFT，是音频模型的标准输入
