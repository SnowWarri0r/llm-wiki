# Qwen-Image-Bench · 文生图评测从"生成"升到"创造"

Qwen 团队 2026 放出的文生图评测基准。论点很直接：模型进步飞快，但**评测拖了后腿**——老基准(GenEval / DPG-Bench)只查"图对不对得上 prompt"(语义对齐),这件事早就饱和了;专业创作者真正在乎的两件事——**像不像真实世界、有没有真创意**——没人系统地量。Qwen-Image-Bench 用"创作者视角"补上,把评测从 generation(生成) 推到 creation(创造)。

## 一句话
**老基准只问"画得对不对",这题已满分;Qwen-Image-Bench 加两根新支柱——真实世界保真 + 创意表达——把模糊的"好不好"拆成 56 条可逐项判定的细则,再训一个跟人类专家 ρ=0.92 对齐的判官自动打分。**

## 它要解决的痛点
- **语义对齐饱和了**:GenEval/DPG-Bench 查"有没有两只猫、红色在不在左边"这类对齐,顶尖模型分数都顶到天花板,**分不出高下**。
- **专业创作要的没人量**:创作者要"这张图能不能当真实产品图用"(真实保真)、"有没有审美和巧思"(创意)。老基准锚在最基础的标准上,这两维空白。
- **"好不好"太主观、判官不可信**:一句"这图好看"没法复现;直接拿一个通用多模态大模型(MLLM)当判官,跟人类口味对不齐。需要**可验证的细则** + **校准过的判官**。

## 核心贡献
1. **五大支柱 taxonomy**:三根老的——质量 / 美学 / 图文对齐;两根新的(创作者驱动)——**真实世界保真**、**创意生成**。后两根是本文的灵魂。
2. **5 → 23 → 56 的层级**:[[rubric-based-evaluation]] —— 5 个支柱拆成 23 个二级子能力、再拆成 **56 条可逐项判定的细则**(每条 0/1/2/N-A 打分),把"好不好"这种模糊判断**拆成一堆能验证的小问题**。完整 23 个二级子能力(56 条三级细则论文未逐条全列):
   - **质量 Quality** (3):真实感 / 细节 / 清晰度
   - **美学 Aesthetics** (6):构图 / 色彩和谐 / 光影 / 人体人像 / 情绪表达 / 风格控制
   - **图文对齐 Alignment** (5):属性 / 动作 / 布局 / 关系 / 场景
   - **真实世界保真 Real-world Fidelity** (3,新):公平性 / 安全合规 / 世界知识
   - **创意生成 Creative Generation** (6,新):想象力 / 特征匹配 / 逻辑自洽 / 文字渲染 / 设计应用 / 视觉叙事
   - 打分聚合:每条细则 0/1/2/N-A,先非线性归一化(Eq.7) `0→0 / 1→60 / 2→100 / N-A 剔除`(及格放 60% 是对齐"60 分及格"直觉),再三级**无权重平均** L3→L2→L1→总分;每条 prompt 只激活 3-5 根支柱,没激活的不进总分。
3. **Q-Judger 判官模型**:[[llm-as-judge]] —— 基于 Qwen3.6-27B,用 **80 位美院专业标注员**盲标(每样本≥3 次独立评审)、13 万+人标对训练出来,自动给 56 条细则打分,跟人类专家排名 **Spearman ρ=0.92**。
4. **1000 条分层双语 prompt**:facet 定向采样(每条至少覆盖 3 个能力点)+ LLM 起草 + 美院艺术家改写 + 长短分半(500 短 500 长)。

## 关键概念

- [[llm-as-judge]] · 训一个判官模型自动打分,校准到人类专家偏好(Q-Judger,ρ=0.92)
- [[image-quality-metrics]] · 对照:FID/CLIPScore/GenEval 那套老指标,本文说它们饱和了

## 我的批注
- 最反直觉也最可信的一点:**这是 Qwen 自己出的基准,但榜首是 GPT Image 2(64.69),Qwen Image 2.0 Pro 只排中游(57.84)**。自家基准自家模型不夺冠,反而说明它没往自己脸上贴金——评测的可信度恰恰来自这种"不护短"。
- "from generation to creation"这个升维是真问题:对齐饱和后,差异全在"真实感"和"创意"上,而这俩最难量化。把它拆成 56 条可判定细则,是把主观审美**工程化**的一次认真尝试。
- 判官的命门是**校准**:不是随便抓个 MLLM 当裁判,而是用 13 万人标 + 80 位美院标注员把判官训到跟专家 ρ=0.92。这跟 [[rlhf]] 训 reward model、[[unlimited-ocr]] 的母题一样——**想要规模化的判断,先用人把判官喂准**。
- GPT Image 2 在五根支柱上**全部第一**(真实保真 57.38、创意 75.23),尤其创意分一骑绝尘——说明顶尖闭源模型的优势已经从"画得对"转到"画得有想法"。
- 聚合方式已查清:不是简单线性,而是 `0→0 / 1→60 / 2→100`(及格故意压到 60% 这条常识线)、N-A 剔除不进分母,再 L3→L2→L1→总分**三级无权重平均**,且每条 prompt 只对激活的 3-5 根支柱算。意味着 N-A 不扣分、单条细则挂 0 会顺着支柱一路放大拖低总分(可定位短板)。
- 待查:双语 prompt 里中英文得分有没有系统性差距。
