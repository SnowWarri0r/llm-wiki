Concept

Anchor 真值分配 · 一个负责，一圈别骂，其他当背景

一句话

每个真值只奖励最佳 anchor；其他高重叠候选忽略，低重叠候选才作为负样本。

直觉

密集检测会在物体附近报出很多相似框。如果只奖励一个最佳框，却把旁边同样盖住物体的框全骂成“背景”，监督会互相打架。Ignore 区的意思是：你没赢，不给奖励；但答案也不算离谱，所以不扣分。

怎么做的


1. 真值框与全部 anchor 中心对齐，形状 IOU 最高者负责
2. 最佳 anchor：objectness=1，并学习坐标与类别
3. 非最佳预测若与任一真值 IOU 超过 ignore threshold：物体性损失记 0
4. 其余预测：objectness=0

坐标与类别损失只落在负责 anchor 上。论文写的忽略阈值是 0.5；官方 YOLOv3 配置使用 0.7。

数字例子

假设一个真值附近有三个候选，最佳 anchor 是 A，忽略阈值取 0.5：

候选 与真值 IOU 身份 物体性监督
A 0.76 最佳 anchor 目标 1，并学习框/类
B 0.62 非最佳但超过阈值 忽略，损失 0
C 0.18 背景候选 目标 0

若当前物体性分数分别为 0.70、0.55、0.20，用平方差只演示监督方向：

A: (1-0.70)² = 0.09
B: ignored = 0
C: (0-0.20)² = 0.04
总计 = 0.13

若错误地把 B 也当背景，还会多出 (0-0.55)²=0.3025，而 B 明明已经较好地盖住物体。✓ Ignore 避免了这条矛盾梯度。
