Concept

K 线买点形态 · 抛压衰竭的几种长相

一句话

缩量下影线 / 箱体底缩量十字星 / 等量反包阳线 / 揉搓线 —— 都在回答同一个问题：在这个震荡区间里，砸盘的还有没有力气了？ 它们是"抛压衰竭、有人接"的不同长相，是缠论横盘龙头低吸、爱在冰川低吸战法里的买点信号。

先拆零件

  • 阳线/阴线：收盘 > 开盘 = 阳线（A 股红、涨）；收盘 < 开盘 = 阴线（绿、跌）。中间粗的是实体（开盘到收盘），上下细线是影线（盘中摸到过但没守住的价）。
  • 缩量 = 今天成交量比前面明显小；等量 = 跟昨天差不多；放量 = 明显放大。量价语言里：缩量 = 没人愿意卖了。

四个买点形态

① 缩量下影线：盘中被砸下去一截，收盘前又被买盘接回来（留长长的下尾巴），而且当天量还缩了 → 想砸的没货了 + 砸下去立刻有人接 = 抛压衰竭 + 有承接。

② 箱体底缩量十字星：箱体 = 价格横向震荡的区间（缠论的中枢/震荡区）；箱体底 = 下沿。十字星 = 开盘≈收盘、实体几乎一条横线 = 多空打平、波动极小。在箱体底冒出一根缩量十字星 → 跌到底抛压枯竭，底就在这。

③ 等量反包阳线：反包 = 今天一根阳线把昨天阴线跌的部分吃回去；等量 = 今天量跟昨天差不多（不是缩量假涨，也不是放天量诱多——放巨量反包反而可能是出货）→ 多方用实打实的承接量重新占上风 = 空转多确认。

④ 揉搓线（术系列，尾盘买）：日线上影接下影（第一天诱多、第二天诱空），尾盘买的好处是次日冲高可立刻止盈、杀跌可及时止损。

串起来 · 都是"空方力竭"的不同长相

  • 缩量下影线 / 缩量十字星 = "没人卖了"（抛压衰竭，偏"等"的信号）
  • 等量反包阳线 = "有人买了"（承接占优，偏"确认"的信号）

两者其实是同一件事——空方力竭、多方接管，对应缠论横盘龙头低吸里「分歧 → 一致」转折的那一刻。

重要 · 形态不能脱离逻辑

爱在冰川反复强调提高成功率的顺序：逻辑 > 预期 > 形态。这些形态只是"扣扳机"的时机信号，前提是这个股本身有强的上涨逻辑（题材/龙头/承接逻辑）。脱离逻辑无脑套形态 = "无脑买买买只会亏亏亏"。所有形态都只有成功率，没有必然。
