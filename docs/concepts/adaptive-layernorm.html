Concept

Adaptive LayerNorm · AdaLN / AdaLN-Zero · 自适应层归一化

一句话

不用固定的归一化缩放/平移，而是用"条件"(时间步 t + 类别/文字 c)临时算出来——让条件去实时拧每一层的旋钮。

直觉

普通 layernorm：归一化之后乘一个学出来的缩放 γ、加一个平移 β，但 γ、β 训练完就固定，每次前向都一样。

可 diffusion-transformer 这类网络有个特殊需求：它必须知道现在去噪到第几步(t)、要生成什么(c)。同一个网络在 t=0.1(刚从噪声起步)和 t=0.9(快成清晰图)该干的事完全不同。怎么把 t、c 喂进去？

AdaLN 的答案：让 γ、β 不再固定，而是用 (t+c) 现算。 一个小 MLP 吃进条件，吐出这一层该用的 γ、β。于是"当前时间和条件"实时拧每层归一化的增益和偏置——网络因此知道自己在轨迹哪、要画啥。比 cross-attention 注入便宜(只动归一化的几个参数)，dit 实测效果还最好。

AdaLN-Zero(DiT 实际用的)再加一手：MLP 除了 γ、β，还为每个残差分支多算一个门控 α(x = x + α·子层(...))，并把 α 初始化成 0。于是开训时 α=0 → 残差分支不起作用 → 整个网络是恒等函数。这跟 resnet "白送 identity 起点"同一招，让深网络训得稳。

怎么做的

先回忆普通 layernorm 三步：

  1. 归一化：向量减自己的均值、除自己的标准差 → 变成均值 0、方差 1 的 x̂
  2. 缩放平移：x̂ ⊙ γ + β（γ、β 是学出来的固定参数，每次前向都一样；⊙ 是逐通道相乘）
  3. 送进下一层

γ 像每个通道的"音量"、β 像"底噪"——固定 LN 把它们焊死了。AdaLN 只改第 2 步参数的来源：γ、β 不再固定，而是一个小 MLP 吃进条件 (t+c) 现算出来。AdaLN-Zero 再加一件：子层(注意力/FFN)的输出先乘一个门控 α 再加回残差，α 也由 MLP 算、且初始化为 0。

cond = embed(t) + embed(c)
γ1,β1,α1, γ2,β2,α2 = MLP(cond)     # 一个小线性层,把条件映成这 6 组数(每块6个)

# —— 注意力子层 ——
x̂ = LayerNorm(x)                   # 第1步:减均值、除标准差
h  = γ1 ⊙ x̂ + β1                   # 第2步:条件在这里拧 x(γ1,β1 来自 cond,不是固定的)
x  = x + α1 ⊙ Attention(h)         # 第3步:α1 控制这块出多少力(α1=0 → 不出力)
# —— FFN 子层同理,用 γ2,β2,α2 ——

γ、β 是"调制"(拧每通道的增益/偏置)，α 是"门控"(决定整块加多少回主干)。

数字例子

拿一个 token 向量 x = [1, 5](2 维，真实是几百维，这里缩小到能手算)，从头算到尾。

第 1 步 · LayerNorm（减均值、除标准差）

均值 = (1+5)/2 = 3
减均值: [1-3, 5-3] = [-2, 2]
方差 = ((-2)² + 2²)/2 = (4+4)/2 = 4 → 标准差 = √4 = 2
x̂ = [-2, 2] / 2 = [-1, 1]              ← 现在均值0、方差1

第 2 步 · 条件算出 γ、β，再调制（AdaLN 的灵魂）

时间步 t=A: MLP 吐出 γ=[2, 0.5], β=[0, 1]
  h = γ ⊙ x̂ + β = [2·(-1)+0,  0.5·1+1] = [-2, 1.5]
       └ 通道1 音量×2、通道2 音量×0.5 还抬了底噪+1

换一个时间步 t=B: MLP 吐出 γ=[0.5, 0.5], β=[0, 0]
  h = [0.5·(-1), 0.5·1] = [-0.5, 0.5]

✓ 同一个 x̂=[-1,1]，t 不同 → h 不同(t=A 得 [-2,1.5]、t=B 得 [-0.5,0.5])。这就是"条件拧每层旋钮"。固定 LN 永远用同一组 γ、β，给不出这种随 t 变化——这正是为什么要 AdaLN。

第 3 步 · 子层 + 零门控 α

设注意力子层在 h 上算出 Attn(h) = [4, -2]
开训 α=0:    x = x + 0 ⊙ [4,-2] = [1, 5]            ← 整块=恒等,x 原样穿过 ✓
训练后 α=0.5: x = x + 0.5 ⊙ [4,-2] = [1+2, 5-1] = [3, 4]   ← 块才开始出力

✓ 自检：α=0 时整块对输入零改动，28 层 DiT 叠起来开训即恒等、梯度好走；训练里 α 慢慢长大块才发力。这就是 AdaLN-Zero 比 vanilla AdaLN(FID 19.47 vs 25.21)强的原因。
