Concept

Adaptive LayerNorm · AdaLN / AdaLN-Zero · 自适应层归一化

一句话

不用固定的归一化缩放/平移，而是用"条件"(时间步 t + 类别/文字 c)临时算出来——让条件去实时拧每一层的旋钮。

直觉

普通 layernorm：归一化之后乘一个学出来的缩放 γ、加一个平移 β，但 γ、β 训练完就固定，每次前向都一样。

可 diffusion-transformer 这类网络有个特殊需求：它必须知道现在去噪到第几步(t)、要生成什么(c)。同一个网络在 t=0.1(刚从噪声起步)和 t=0.9(快成清晰图)该干的事完全不同。怎么把 t、c 喂进去？

AdaLN 的答案：让 γ、β 不再固定，而是用 (t+c) 现算。 一个小 MLP 吃进条件，吐出这一层该用的 γ、β。于是"当前时间和条件"实时拧每层归一化的增益和偏置——网络因此知道自己在轨迹哪、要画啥。比 cross-attention 注入便宜(只动归一化的几个参数)，dit 实测效果还最好。

AdaLN-Zero(DiT 实际用的)再加一手：MLP 除了 γ、β，还为每个残差分支多算一个门控 α(x = x + α·子层(...))，并把 α 初始化成 0。于是开训时 α=0 → 残差分支不起作用 → 整个网络是恒等函数。这跟 resnet "白送 identity 起点"同一招，让深网络训得稳。

怎么做的

条件 = embed(t) + embed(c)
  ↓ 小 MLP
每个 DiT 块拿到 6 个向量: (γ1,β1,α1) 给注意力子层, (γ2,β2,α2) 给 FFN 子层

注意力子层:  h = LayerNorm(x);  h = γ1 ⊙ h + β1;  x = x + α1 ⊙ Attn(h)
FFN 子层:    h = LayerNorm(x);  h = γ2 ⊙ h + β2;  x = x + α2 ⊙ FFN(h)

α 零初始化 → 开训 x=x；训练中 α 慢慢长大，块才开始"出力"。

数字例子

某一层归一化后得到 x̂ = [1, -1](均值 0、标准差 1)。

缩放/平移：固定 LN 用 γ=1, β=0 → 输出还是 [1, -1]。AdaLN 在某个 (t,c) 下 MLP 算出 γ=2, β=0.5：

out = γ ⊙ x̂ + β = 2·[1,-1] + 0.5 = [2.5, -1.5]

换一个时间步 t'，MLP 可能算出 γ=0.5, β=0 → [0.5, -0.5]。✓ 同一个 x̂，不同 t 给出不同调制——这就是"条件拧旋钮"。

零门控的恒等起点：设注意力子层算出 Attn(h) = [3, 3]。

开训 α1=0:   x = x + 0·[3,3] = x        ← 块=恒等,信号原样穿过 ✓
训练后 α1=0.2: x = x + 0.2·[3,3] = x+[0.6,0.6]  ← 块开始出力

✓ 自检：α=0 时整块对输入零改动，28 层 DiT 叠起来开训即恒等，梯度好走；这就是 AdaLN-Zero 比 vanilla AdaLN(19.47 vs 25.21 FID)强的原因。

跟其它条件注入的对比

怎么注入条件 代价 DiT 里 FID(@400K)
