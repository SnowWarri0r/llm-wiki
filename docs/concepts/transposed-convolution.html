Concept

Transposed Convolution · 转置卷积 · 可学习的上采样

一句话

普通卷积把一片输入汇成较少输出；转置卷积把每个输入值按可学习卷积核“铺”到更大的输出网格，并把重叠贡献相加。

先纠正名字

它不是把普通卷积的结果精确还原，也不是真正的逆卷积。名字里的“转置”来自线性代数：把普通卷积写成矩阵乘法 y=Ax 后，转置卷积执行形状相反的 Aᵀy。

怎么做的

最小一维例子，输入 [2,3]，核 [1,2]，stride=1：

输入 2 铺出：2×[1,2] = [2,4,0]
输入 3 铺出：3×[1,2] = [0,3,6]
重叠相加：              [2,7,6]

输出从长度 2 变成 3。二维同理：每个输入位置把一个 k×k 小块写进更大的平面，重叠处求和。

数字例子 · stride=2 怎样放大

输入 [2,3]，核仍为 [1,2]，stride=2：

2 的小块从位置 0 开始：[2,4,0,0]
3 的小块从位置 2 开始：[0,0,3,6]
相加：                 [2,4,3,6]

输出长度翻到 4。真实 U-Net 用二维 2×2 上采样把宽高各放大 2 倍。

它和插值有什么差别

  • 最近邻 / 双线性插值：放大规则固定，没有可学习参数；
  • 转置卷积：放大核由数据学；
  • 插值 + 普通卷积：先固定放大，再学习如何混合，常用来减少棋盘格伪影。

上采样只负责增加空间位置，不能凭空恢复池化前的精确边界。U-Net 仍需要跳跃连接补回高分辨率特征。
