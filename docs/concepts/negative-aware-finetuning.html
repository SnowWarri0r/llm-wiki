Concept

负样本感知微调 · NFT · 用奖励切正负,在前向过程上把 RL 折成监督损失

一句话

给每张生成图打个奖励 r∈[0,1],按 r 把样本软切成正堆(高奖励)和负堆(低奖励);让一个模型同时学"正速度"和"负速度",正负之差自然指向"该往哪改"——于是扩散的在线 RL 退化成一个加权的监督 flow-matching 损失,不碰似然、不碰反向采样轨迹。

直觉 · 为什么要绕开反向、回到前向

扩散做 RL 卡在似然算不出来(diffusion-rl-likelihood-barrier)。主流绕法(FlowGRPO)是把反向去噪切成多步马尔可夫过程让概率可算,再套 grpo——但反向路有无数条,盯着它训会锁死采样器、还可能让模型偏离真正的前向过程。

NFT 的洞察一句话:前向加噪只有一条、反向去噪有无数条。既然前向是唯一且良定义的,那就把 RL 做在前向上。前向过程正是 flow-matching 的训练地盘——所以整件事能接回最朴素的监督训练,几乎不改代码。

怎么做的 · 奖励软切 + 单模型隐式装正负


1. 用任意 solver(ODE/SDE/高阶都行)采一批干净图 x₀,各打奖励 r∈[0,1]
2. 概念上切两堆:
     正分布 π⁺ ∝ r · π_old        (高奖励样本权重大)
     负分布 π⁻ ∝ (1−r) · π_old    (低奖励样本权重大)

3. 一个 velocity 网络 v_θ 隐式同时表示正负两副动态(围绕冻结的旧模型 v_old):
     v_θ⁺ = (1−β)·v_old + β·v_θ      (正/被引导的速度)
     v_θ⁻ = (1+β)·v_old − β·v_θ      (负速度)   ← 两者对称于 v_old

4. 监督损失(纯 flow matching,不是 policy gradient):
     L = E[ r·‖v_θ⁺ − v‖² + (1−r)·‖v_θ⁻ − v‖² ]
   高奖励样本(r→1)把 v_θ⁺ 拉向它的真速度、低奖励(r→0)把 v_θ⁻ 拉向它的真速度

正负只要存干净图就能算,不用整条采样轨迹;r vs 1−r 的对比权重 + 隐式参数化,把强化信号揉进了普通的监督目标。

数字例子 · 隐式参数化对称在 v_old 周围

取标量简化(真实是逐维速度向量):旧模型速度 v_old=1.0、模型当前输出 v_θ=1.4、引导强度 β=0.5。

v_θ⁺ = (1−0.5)·1.0 + 0.5·1.4 = 0.5 + 0.7 = 1.2     (正速度)
v_θ⁻ = (1+0.5)·1.0 − 0.5·1.4 = 1.5 − 0.7 = 0.8     (负速度)

自检①:v_θ⁺ + v_θ⁻ = 1.2 + 0.8 = 2.0 = 2·v_old   → 正负永远对称夹着 v_old ✓
自检②:v_θ⁺ − v_θ⁻ = 0.4 = 2β·(v_θ − v_old) = 2·0.5·0.4 ✓  → 这个缝隙 = 模型离旧模型多远

✓ 含义:模型只动一个量 v_θ,正速度往上抬到 1.2、负速度对称压到 0.8。训练时高奖励样本把"上抬的那副"(v_θ⁺)拉准、低奖励把"压下的那副"(v_θ⁻)拉准,两股梯度合起来,就把 v_θ 推向"正比负更该走"的方向。采样时只用 v_θ⁺(被引导的正速度)——等于把 reinforcement-guidance 内建进了一个模型,不再需要 CFG 的两套模型。最优解 v_θ* = v_old + (2/β)·Δ,小小的改进方向 Δ 被 2/β 放大着学进去。

最优解从哪来 · 把损失对 v_θ 求导置零

v_θ* = v_old + (2/β)·Δ 不是凑的,是把损失这个"碗"找到碗底解出来的。下面全程拿两个样本带数字走一遍,不用记任何期望公式。

先搭舞台:期望就是"加权平均"。 把 E[X] 读成"X 在这批样本上的平均值",只要两条规则:(规则1)对每个样本都一样的量是常数,能拎到平均号外;(规则2)平均一个和/差,等于各自平均的和/差。关键是分清谁是常数、谁是变量:v_θ⁺ / v_θ⁻ / v_old 是模型在某个点 x_t 的输出,对所有样本同一个值 = 常数;只有奖励 r 和目标速度 v 是逐样本不同的变量。

设落在这点的就两条样本,奖励用 0/1 最直观(好样本进"正桶"、差样本进"负桶"):

样本 奖励 r 目标速度 v 进哪桶
A 1 1.4 正桶
B 0 0.6 负桶

几个符号其实都只是某种平均,先坐实:

ρ⁺ = E[r]   = (1+0)/2     = 0.5    好样本占比
ρ⁻ = E[1−r] = (0+1)/2     = 0.5    差样本占比
v_old = E[v] = (1.4+0.6)/2 = 1.0    所有样本速度的平均
v⁺ = 正桶里 v 的平均 = 1.4(只有 A)
v⁻ = 负桶里 v 的平均 = 0.6(只有 B)

① 损失是个碗,碗底=斜率为 0。 单样本损失 ℓ = r(v_θ⁺−v)² + (1−r)(v_θ⁻−v)²,总损失 L=E[ℓ] 是开口朝上的碗(平方和)。找最优 v_θ = 找碗底,而碗底那点切线是平的、斜率(导数)= 0。所以接下来就一件事:算出 L 对 v_θ 的斜率,令它等于 0。

求斜率用平方求导规则 d(东西)²=2·东西·东西的导数。注意 v_θ⁺=(1−β)v_old+β·v_θ 对 v_θ 的导数是 +β、v_θ⁻=(1+β)v_old−β·v_θ 的导数是 −β:

dℓ/dv_θ = r·2(v_θ⁺−v)·(+β) + (1−r)·2(v_θ⁻−v)·(−β)
        = 2β·[ r(v_θ⁺−v) − (1−r)(v_θ⁻−v) ]

负项带的 −β 翻出来就是中间那个减号。两股劲是同向叠加的(正样本抬正速度、负样本压负速度,都把 v_θ 往同一边推),这个叠加就是最后 2/β 里那个 2 的来源。

② 把斜率设成 0,得到平衡式 (★)。 总斜率 = 各样本斜率的平均 E[dℓ/dv_θ]=2β·E[ r(v_θ⁺−v) − (1−r)(v_θ⁻−v) ]。令它 = 0,2β 非零除掉,再用规则 2 把减法拆开移项:

E[r(v_θ⁺−v)] = E[(1−r)(v_θ⁻−v)]      …(★)

这就是"碗底斜率为零"的另一种写法,意思是好样本把正速度拉准的劲,跟差样本把负速度拉准的劲,刚好扯平。

它两边还能各自化简。看左边,用规则1把常数 v_θ⁺ 拎出来(这是初中的提公因数):

E[r(v_θ⁺−v)] = E[r·v_θ⁺] − E[r·v] = v_θ⁺·E[r] − E[r·v] = ρ⁺·v_θ⁺ − ρ⁺·v⁺ = ρ⁺(v_θ⁺−v⁺)

其中 E[r·v]=ρ⁺·v⁺:r 是 0/1 时,r·v 把差样本清零、只留好样本的 v,平均下来 = 好样本占比 × 好样本平均速度。数字验:E[rv]=(1·1.4+0·0.6)/2=0.7 = 0.5·1.4 = ρ⁺v⁺ ✓。右边同理 = ρ⁻(v_θ⁻−v⁻),于是 (★) 就是 ρ⁺(v_θ⁺−v⁺)=ρ⁻(v_θ⁻−v⁻)。

把这个碗直接画出来感受一下(β=0.5、v_old=1.0,故 v_θ⁺=0.5+0.5v_θ、v_θ⁻=1.5−0.5v_θ;A 只有正项、B 只有负项):

L = [ (v_θ⁺−1.4)² + (v_θ⁻−0.6)² ]/2 = (0.5v_θ − 0.9)²     ← 一条标准抛物线
碗底在 0.5v_θ−0.9=0 → v_θ=1.8（就是最优解 ✓）
查 (★) 在不同 v_θ 平不平衡（左=v_θ⁺−1.4，右=v_θ⁻−0.6）:
   v_θ=1.0:  左=1.0−1.4=−0.4   右=1.0−0.6=+0.4   不等 → 没到碗底
   v_θ=1.8:  左=1.4−1.4= 0      右=0.6−0.6= 0      相等 → 碗底,斜率=0 ✓

③ 两个一眼能信的事实。

  • 事实 A:v_old = ρ⁺v⁺ + ρ⁻v⁻。 就是"总平均 = 两桶各自平均的加权",跟"班级均分 = 男生均分×男生比例 + 女生均分×女生比例"一模一样。数字:0.5·1.4 + 0.5·0.6 = 1.0 = v_old ✓。
  • 事实 B:改进方向 Δ = ρ⁺ρ⁻(v⁺−v⁻)。 本质是"从负桶指向正桶"的方向 v⁺−v⁻(往高奖励那边),前面 ρ⁺ρ⁻ 只是缩放。由事实 A 挪一下项可得它等于 ρ⁺(v⁺−v_old):因为 v⁺−v_old = v⁺−(ρ⁺v⁺+ρ⁻v⁻) = ρ⁻(v⁺−v⁻)(用 1−ρ⁺=ρ⁻),两边乘 ρ⁺ 即得。数字:Δ = 0.5·0.5·0.8 = 0.2,也 = 0.5·(1.4−1.0) = 0.2 ✓。

④ 代回 (★),解出 v_θ*。 把参数化 v_θ⁺=(1−β)v_old+β·v_θ、v_θ⁻=(1+β)v_old−β·v_θ 代进 (★),归拢同类项(用 ρ⁺+ρ⁻=1),中括号那坨用事实 A 一化简,恰好就是 2Δ:

β·v_θ* = β·v_old + [ρ⁺v⁺ − ρ⁻v⁻ − (ρ⁺−ρ⁻)v_old] = β·v_old + 2ρ⁺ρ⁻(v⁺−v⁻) = β·v_old + 2Δ
两边除以 β  ⟹  v_θ* = v_old + (2/β)·Δ

2/β 在说什么。 2 = 上面那两股梯度同向叠加;1/β = 模型偏离 v_old 的部分只按 β 倍"漏进"被引导速度(因为 v_θ⁺=v_old+β(v_θ−v_old)),所以想让引导速度体现一个改进量 Δ,模型本身得学一个放大 1/β 倍的偏移——β 越小,v_θ 偏得越狠。

整段用数字收口: ρ⁺=0.5、v⁺=1.4、v⁻=0.6 → v_old=1.0、Δ=0.2;β=0.5 → v_θ* = 1.0 + (2/0.5)·0.2 = 1.0 + 4·0.2 = 1.8。回代验证被引导速度:v_θ⁺ = (1−0.5)·1.0 + 0.5·1.8 = 1.4 = v⁺ ✓(奖励对半时,最优的被引导速度正好复现正桶速度,合理)。
