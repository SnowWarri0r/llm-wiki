LPIPS ·
像不像，让网络说了算。

Zhang · Isola · Efros · Shechtman · Wang · CVPR 2018 · arXiv 1801.03924

逐像素比两张图（L2/PSNR）跟人眼差得离谱：平移一像素 L2 暴增却看着没变，糊成一团 L2 反而小。LPIPS 的答案：把两张图都喂进一个预训练网络、比深层特征——深度特征作为"像不像"的尺子，好得不讲道理。

TL;DR

LPIPS（Learned Perceptual Image Patch Similarity）= 在预训练网络的特征空间里量"两张图像不像"1。把两图过一个 frozen 的 CNN（VGG/AlexNet），取多个层的特征、通道单位归一化、乘一组学出来的线性权重2，算 L2、空间平均、各层求和。那组权重是拿 BAPPS 数据集3 的人类 2AFC 判断4（"这两个哪个更像参考？"）校准的。核心发现——深层特征作为感知度量好得"不讲道理"5，几乎任何预训练网络都远胜 L2/SSIM，说明感知相似是深度表征的涌现属性。既是度量，也常被当训练损失（见 perceptual-loss）。

§ 02怎么算 · 深层特征 + 学出的权重

LPIPS 不在像素上量，在预训练网络的特征空间里量。两张图都过同一个 frozen 网络，取若干层的特征图对比——再用一组学出来的权重把各通道/各层加起来。

§ 03权重哪来 · 人类 2AFC 校准

那组线性权重不是拍脑袋，是拿人类判断校准的。作者建了 BAPPS 数据集：给人看一个参考图 + 两个失真版本，问"哪个更像参考？"（这叫 2AFC，二选一强制选择）。收一堆人的票，再调 LPIPS 的权重让它的打分跟人投的票一致。

§ 04"不讲道理的好" · 当度量也当损失

论文标题就是结论：The Unreasonable Effectiveness of Deep Features——深层特征当感知度量好得"不讲道理"。最反直觉的发现：几乎任何预训练网络的特征（VGG / AlexNet / SqueezeNet，甚至自监督、甚至随机初始化都比像素强）都远胜 PSNR/SSIM。说明"感知相似"是深度视觉表征自带的涌现属性，不是某个特定网络的功劳。

LPIPS 既是度量（比 PSNR/SSIM 更准地评图像质量），也是损失：生成模型训练时把"预测图 vs 真值图"的 LPIPS 距离加进 loss，逼输出在感知上像，而不是像素上像。HiDream-O1 就拿 LPIPS + DINO perceptual loss 一起当附加监督。要点：把"像不像"从像素空间挪到一个懂图的网络的特征空间——跟 perceptual loss 是一回事，LPIPS 是它最经典的实例。

引用 / 术语表

  1. LPIPS · Learned Perceptual Image Patch Similarity。在预训练网络特征空间里量两图距离的感知度量/损失，越低越像。详见 perceptual-loss。
  2. 学出的线性权重 · 取多层特征、通道单位归一化后，每通道乘一个学出来的权重再 L2 求和。网络冻结，只有这组很轻的权重是训出来的——所以叫 "learned"。
  3. BAPPS 数据集 · Berkeley-Adobe Perceptual Patch Similarity。作者收集的人类感知判断数据，用来训练/评估感知度量。
  4. 2AFC · 二选一强制选择 · 给参考图 + 两个失真版，问人"哪个更像参考"。比让人打绝对分更稳。LPIPS 的权重就拿这些票校准。
  5. unreasonable effectiveness · 论文核心发现：几乎任何预训练网络的深层特征当感知度量都远胜 L2/SSIM → 感知相似是深度表征的涌现属性，不依赖特定网络。
