LSTM ·
给 RNN 修一条记得住远处的传送带。

Hochreiter & Schmidhuber · Long Short-Term Memory · Neural Computation · 1997

普通 RNN 一步步往后传隐藏状态，梯度连乘几十步就消失，记不住长程依赖。LSTM 加一条单独的"记忆传送带"（cell state），更新主要靠加法、信息能原样带很远；再配三个阀门（遗忘/输入/输出）决定擦掉什么、写入什么、读出什么。Transformer 之前的序列霸主。

TL;DR

普通 RNN 反向传播穿过多步 → 梯度连乘 → 消失/爆炸1，学不到长程依赖。LSTM 加一条单独的 cell state（记忆传送带 C）2，更新是加法 C_t = f·C_{t-1} + i·g → 信息和梯度都能带很远。三个门（0~1 的阀门）：遗忘门 f3（老记忆留多少）、输入门 i + 候选 g4（写什么新的）、输出门 o5（读出什么当 h_t = o·tanh(C_t)）。梯度不消失的根：加法路径 ≈ 每步乘 1，同 ResNet 残差 x+F(x)6（LSTM 1997 就用了，ResNet 2015 在深度方向重新发明）。简化版 GRU（2 门）。2014–2017 序列霸主（seq2seq），最早的 attention 加在它上面；后来 Attention Is All You Need 把循环全扔了取代它。

§ 01痛点：RNN 记不住远处

普通 RNN 把序列一步步处理，拎着一个隐藏状态 h 往后传，每步 h_t = tanh(W·[h_{t-1}, x_t])。反向传播要穿过几十上百步，梯度沿途连乘同一批权重和导数：

  • 连乘的数 <1 → 越乘越小 → 梯度消失（远处的信号传不回来，学不到长程依赖）。
  • 连乘的数 >1 → 越乘越大 → 梯度爆炸。

后果："那只猫……（隔 50 个词）……是黑色的"——等要让"是"跟"猫"呼应时，信号早被乘没了。RNN 实际只记得住几步。

§ 02核心：一条记忆传送带 + 三个阀门

LSTM 加一条单独的 cell state（细胞状态 C）——像一条传送带 / 记忆高速路从头通到尾，中间只有很轻的线性改动。关键：更新 C 主要靠加法（不是反复乘权重矩阵），信息能原样带很远。三个门（各是一个 sigmoid 出 0~1 = "放多少通过"的阀门）控制这条带。

f_t = σ(W_f·[h_{t−1}, x_t])        # 遗忘门：老记忆留多少（0~1）
i_t = σ(W_i·[h_{t−1}, x_t])        # 输入门：新的写多少
g_t = tanh(W_g·[h_{t−1}, x_t])     # 候选：要写的内容
C_t = f_t · C_{t−1} + i_t · g_t    # 传送带更新：擦一点 + 加一点（加法！）
o_t = σ(W_o·[h_{t−1}, x_t])        # 输出门：读出多少
h_t = o_t · tanh(C_t)              # 这一步的隐藏状态/输出

§ 04为什么梯度不消失：加法高速路

关键在 cell state 的更新是加法：C_t = f_t · C_{t−1} + i_t · g_t。反向传播沿 C 这条路往回走，梯度每步大约只乘一个遗忘门 f（≈1），而不是普通 RNN 那样乘一个会缩水的"权重×tanh导数"。所以记忆和梯度都能流很远——传送带是给梯度修的一条高速路。

眼熟吗

这跟你学的 ResNet 残差 x + F(x) 是同一个魂：给梯度修一条加法直通路，让它绕过会缩水的非线性。LSTM 1997 年就在时间方向这么干了；ResNet 2015 在深度方向重新发明同一招。"加法路径 = 梯度高速路"是深度学习反复出现的母题。

§ 05余波：GRU · 被 Transformer 取代

GRU（Gated Recurrent Unit）：LSTM 的简化版——只有 2 个门（更新门 + 重置门）、把 cell state 和 hidden state 合一。更省参数、训练更快，很多任务上跟 LSTM 打平。

被 Transformer 取代：LSTM 是 2014–2017 的序列霸主，seq2seq 机器翻译就是 LSTM 编码器→解码器，而最早的 attention 是加在 LSTM 上的（让解码器回看编码器各步）。然后 Attention Is All You Need 干脆把循环全扔了：

LSTM（循环）Transformer（注意力）
长程依赖靠传送带一步步传任意两位置直接互看
并行难（必须按时间顺序算）易（整条序列一起算）
路径长度O(序列长)O(1)

所以 LSTM 正是 Transformer 革掉的那个前任——但"门控 + 加法记忆通路"的思想留了下来，到处都是。

放进 wiki 这条线

LSTM = 序列建模这条线的起点：门控阀门 + 加法记忆传送带治住 RNN 的梯度消失。加法梯度高速路 = ResNet 残差同构；它撑起 seq2seq 和最早的 attention，最终被 Transformer 用"任意位置直接互看 + 可并行"取代。一句话：RNN 记不住远处；LSTM 修一条加法传送带 + 三个阀门，让记忆和梯度都能流很远。
