FIELD NOTES · 文生图评测

从生成
到创造

Qwen Team, Alibaba · 2026 · arXiv 2605.28091

老基准只查"图对不对得上 prompt"，这题早满分了。Qwen-Image-Bench 加两根新支柱——真实世界保真 + 创意——把模糊的"好不好"拆成 56 条可判定细则。

TL;DR

文生图模型进步飞快，评测却拖了后腿。老基准（GenEval / DPG-Bench）只查语义对齐——"有没有两只猫、红色在不在左边"，顶尖模型都顶到天花板、分不出高下。专业创作者真正在乎的两件事没人系统地量：像不像真实世界、有没有真创意。Qwen-Image-Bench 用创作者视角补上：五大支柱（质量/美学/图文对齐 + 真实世界保真 + 创意生成），按 5→23→56 拆成 56 条可逐项判定的细则，再训一个跟人类专家 ρ=0.92 对齐的判官 Q-Judger 自动打分。结果很坦诚——自家基准里榜首是 GPT Image 2（64.69），Qwen 自己的模型只排中游。

§ 01评测要从"生成"升到"创造"

这几年文生图模型一代比一代强，可衡量它们的尺子还停在老地方。老基准查的是语义对齐9——画面里该有的东西有没有、位置颜色对不对。这题顶尖模型早就做到接近满分，饱和了，分不出高下。但专业创作者真正挑剔的两件事，老基准压根没量：这张图像不像能用的真实产品图（真实保真），以及有没有审美和巧思（创意）。Qwen-Image-Bench 的主张就是把评测从 generation（画得对）推到 creation（画得有想法）。

§ 02五大支柱：三根老的，两根新的

Qwen-Image-Bench 把"图好不好"分成五根支柱。三根是老熟人——质量、美学、图文对齐；两根是创作者驱动的新维度——真实世界保真2（能不能当真实产品图用）和创意生成3（有没有审美巧思）。后两根才是本文的灵魂，也是老基准的空白。

§ 03把"好不好"拆成 56 条可判定细则

"这图好不好"直接打 1-10 分，主观、难复现。Qwen-Image-Bench 用 rubric（细则）把它拆成一棵树：5 根支柱 → 23 个二级子能力 → 56 条三级细则4，每条只判 0/1/2/N-A（不满足/部分/满足/不适用）。逐条打分再聚合——既可复现，又能告诉你模型具体差在哪条。详见 rubric-based-evaluation。

全部 5 支柱 → 23 子能力（一张表看全）

56 条三级细则论文没逐条全列（只给了部分名字：光影物理、材质质感、噪点、边缘清晰度……），但二级的 23 个子能力是完整公开的，挂在 5 根支柱下面如下。带 NEW 的两根（真实世界保真 / 创意生成）是创作者驱动的新维度，也是这篇基准的灵魂；其余三根是老基准就有、如今已饱和的部分。

每条怎么打分、怎么往上滚成总分

每条细则判 0/1/2/N-A，但聚合前先做一次非线性归一化（论文 Eq.7），再三级求平均：

分数映射（关键：不是线性 0/50/100） 0 不满足 → 0 1 及格 → 60 ← 故意把"及格"放在 60% 这个常识线 2 优秀 → 100 N-A 不适用 → 剔除 ← 不算分母,既不拉低也不抬高 往上滚（全程无权重平均,且只算"这条 prompt 实际激活的"子项） 56 条细则 →(同一子能力内求平均)→ 23 子能力分 23 子能力 →(同一支柱内求平均)→ 5 支柱分 5 支柱 →(激活的那几根求平均)→ 总分 注:每条 prompt 只激活其中 3-5 根支柱(画风景不考"人体", 抽象画不考"世界知识"),没激活的支柱不进总分。

把"及格 = 60"而不是 50，是为了对齐"60 分及格"的直觉——一张"没毛病但也不出彩"的图，落在 60 才符合人对"刚及格"的感觉。

拿一张图走一遍（prompt 激活了 3 根支柱），先把每条细则的 0/1/2 换成 0/60/100，N-A 直接剔除：

【真实世界保真】 子能力·物理合理: 光影方向 2→100 反光高光 1→60 → (100+60)/2 = 80 子能力·材质: 纹理真实 2→100 防伪水印 N-A(剔除) → 100/1 = 100 支柱分 = (80 + 100) / 2 = 90 【质量】 子能力·真实感 1→60 子能力·细节 2→100 → 各 1 条 支柱分 = (60 + 100) / 2 = 80 【创意生成】 子能力·想象力: [2,1]→[100,60] → (100+60)/2 = 80 子能力·文字渲染: 0→0 → 0 支柱分 = (80 + 0) / 2 = 40 总分 = 激活的 3 根支柱无权重平均 = (90 + 80 + 40) / 3 = 70.0

✓ 自检：N-A 那条（防伪水印）不进分母，所以"材质"是 100 不是 50——不适用不等于扣分。"文字渲染"挂了个 0，直接把创意这根支柱从 80 拖到 40，再压低总分——这正是拆细则的价值：总分 70 能一路回溯到"这模型栽在文字渲染上"，而不是只甩一个笼统的 70。

§ 04Q-Judger：把判官训到跟人类对齐

56 条细则要逐张图判，靠人评 1000 条 prompt×几十个模型太贵太慢。自然想到让模型当判官。但随便抓个通用 MLLM 当裁判，口味跟人类专家对不齐。关键是校准：用 80 位美院标注员盲标（每样本≥3 次独立评审）、13 万+人标对训练，把判官（基于 Qwen3.6-27B）调到跟人类专家排名 Spearman ρ=0.92，才敢放它自动评。详见 llm-as-judge。

§ 05结果：自家基准，自家模型不夺冠

最能说明这基准没护短的一点：榜首是 GPT Image 2（64.69），在五根支柱上全部第一（真实保真 57.38、创意 75.23 一骑绝尘）；Qwen 自己的 Qwen Image 2.0 Pro 只排中游（57.84）。自家出的考卷自家学生不拿第一，反而让这份评测更可信。

放进 wiki 这条线

Qwen-Image-Bench 是给文生图模型出的"考卷"——Qwen-Image-2.0 / SD3.5 / FLUX（主干都是 DiT）是被考的"考生"。它的两个方法骨架都可复用:拆细则打分(把模糊的好不好拆成可验证小问题)、训校准的判官(跟 RLHF 的 reward model 同母题)。它要超越的是 FID/CLIPScore/GenEval 那套已饱和的老指标。一句话:对齐做满之后,模型的差距全在"真实感"和"创意"上;把这两维拆成 56 条能逐项判定的细则、再训个跟人类 ρ=0.92 的判官自动打,评测才追得上模型——而自家基准让对手夺冠,正是它可信的证据。

引用 / 术语表

  1. 五大支柱 · 质量 / 美学 / 图文对齐（传统 3 根，已饱和）+ 真实世界保真 / 创意生成（创作者驱动的 2 根新维度，本文灵魂）。
  2. 真实世界保真 · 能不能当真实产品图用：光影物理、材质质感、有无常识穿帮。老基准的空白之一。
  3. 创意生成 · 有没有审美和巧思（不只是"画对了"）。GPT Image 2 在这维 75.23 一骑绝尘。
  4. 5 → 23 → 56 层级 · 5 支柱拆成 23 个二级子能力、再拆成 56 条三级细则。把模糊判断拆成可验证的树。
  5. 可判定细则 · rubric · 每条细则只判 0/1/2/N-A（不满足/部分/满足/不适用），逐条打分再聚合 → 可复现、能定位短板。
  6. Q-Judger · 基于 Qwen3.6-27B 的判官模型，自动给 56 细则打分。靠大量人标校准到人类口味才可信。
  7. Spearman ρ=0.92 · 判官排名跟人类专家排名的等级相关（1=完全一致）。0.92 = 几乎一样 → 判官可信。
  8. GPT Image 2 = 64.69 · 榜首，五支柱全部第一。Qwen 自家 Qwen Image 2.0 Pro 仅 57.84（中游）。
  9. 语义对齐饱和 · GenEval/DPG-Bench 查"画面元素对不对"，顶尖模型都接近满分、分不出高下，逼出"升维到创造"的需求。
  10. 1000 双语 prompt · facet 定向采样（每条≥3 能力点）+ LLM 起草 + 美院艺术家改写 + 长短分半（500 短 500 长）。
