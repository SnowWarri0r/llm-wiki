不画一张拍平的图，生成一摞图层。

Canva Research · MRT: Masked Region Transformer for Layered Image Generation and Editing at Scale · CVPR 2026 · arXiv 2605.27235

拍平的图一旦生成就难改——挪个 logo、换个元素就得整张重画。MRT 直接产出一摞可编辑的 RGBA 图层（画布+背景+前景），每层透明、能单独挪改重组。妙在：同一个模型靠"哪些图层是干净给定、哪些是噪声待生成"这一个开关，就切出文生层 / 拆图成层 / 层改层三种任务。

TL;DR

MRT（20B，建在 Qwen-Image 上）做分层图像生成与编辑：产出可编辑的 RGBA 图层1（画布+背景+K 前景），不是拍平图。核心一招——选择性 token masking2：哪些图层从干净 latent（给定、不预测）初始化、哪些从噪声（待生成）初始化，这个开关本身就是任务：全噪声=文生层；合成图干净/图层噪声=拆图成层；已有层干净/新层噪声=层改层。干净↔噪声 token 间 full attention。这就是 Qwen-Image-2.0 "条件→目标"的升级版（任意子集当条件）。Anonymous Region Transformer3：每层裁剪成 region token（WAN-2.1-VAE），不给身份标签、靠位置+内容推角色 → 可任意重摆。Overflow 画布层4留住溢出边界的像素（60%+ 设计有溢出）。DMD 蒸馏5 50→8 步、6× 提速、~20 层峰值 108.5×。全参数微调、10M+ 多语设计。比 Qwen-Image-Layered 快 10~100×、省 50~90% 显存。

§ 01痛点：拍平的图改不动

普通文生图产出一张拍平（flat）的图：所有元素糊在一个像素平面上。想挪个 logo、换句文案、重排版？——只能整张重画，没法只动一处。

设计真正要的是分层：画布、背景、每个前景元素各是一层、透明可叠。分层 = "图像版的逐词编辑"——每个元素能单独挪/改/重组，不重生成整张，是可复用的设计素材。

§ 02核心：masking 哪些干净/哪些噪声 = 切任务

MRT 是个在图层 token 上跑的扩散模型。最妙的一手：你控制哪些图层从"干净 latent（给定、不预测）"开始、哪些从"噪声（要生成）"开始——这个开关本身就是任务。一个 20B 模型，翻翻 mask 就干三件事：

这正是 Qwen-Image-2.0 "条件→目标、编辑=条件里塞原图"的升级版：不再只塞一张原图，而是任意一组图层都能当"干净条件"、其余当"噪声目标"。

一个类比

一摞拷贝纸：告诉模型哪几张已经画好（干净）、哪几张空白（噪声待填）。全空白 = 从零拼整摞；只有最底那张平图画好、其余空白 = 把它拆成层；画好几张、留一张空 = 加一张配套的层。

§ 03Anonymous Region Transformer

每个图层先被裁剪成一块区域，用 WAN-2.1-VAE 编成各自的 region token；所有前景层 token + 背景层 token + 合成图 token 一起做 full attention（生成的层能看见给定的层，层间一致）。

"匿名（anonymous）"的意思：不给图层身份标签，模型纯靠每块区域的空间位置 + 视觉内容推断它是什么角色。好处是每个前景层都保留完整空间信息 → 能任意重新摆位，复用性拉满。

前景层₁(裁剪) ┐
前景层₂(裁剪) ├─ WAN-2.1-VAE → region tokens ─┐
背景层        │                                ├─ full attention（全部一起）→ 去噪
合成全图      ┘                                ┘   干净 token 与噪声 token 互看

底座是 Qwen-Image（~20B，60 层 / hidden 3584 / 24 头）。因为"平图→分层"分布变化大，做全参数微调（不是只调 LoRA），FSDP2 训。

§ 04Overflow 画布：别在边界把图层切了

以前的分层做法在画布边界把图层截断——超出可见区的像素直接丢。结果你想把元素往外挪一点再挪回来，它缺了一块，不可编辑。MRT 加一个全尺寸透明画布层，把溢出部分也完整留住（训练集 60%+ 的设计有溢出层）。

§ 05蒸馏 · 结果 · 放进 wiki

DMD 蒸馏：50 步 → 8 步

跟 Qwen-Image-2.0 同一个 DMD（Distribution Matching Distillation）：最小化师生转移分布的 KL（去噪每步是 高斯马尔可夫 转移），把 50 步老师压成 8 步学生，6× 提速、FID 仅 16.02→18.58，H200 上 ~2.3s 实时；~20 层时峰值提速 108.5×。同 Drifting/DiffusionOPD 的少步化线。

引用 / 术语表

  1. RGBA 图层 · 带透明通道(Alpha)的图像层：画布层(全尺寸透明底) + 背景层(半透明) + K 个前景层。可单独挪/改/重组 = "图像版逐词编辑"。
  2. 选择性 token masking · 每个图层从干净 latent(给定、不预测)还是噪声(待生成)初始化。这个开关本身就是任务：全噪声=文生层 / 合成图干净=拆图成层 / 已有层干净=层改层。干净↔噪声 full attention。
  3. Anonymous Region Transformer · 每层裁剪成 region token(WAN-2.1-VAE)，所有前景/背景/合成 token 一起 full attention。"匿名"=不给身份标签、靠位置+内容推角色 → 每层留完整空间信息、可任意重摆。
  4. Overflow 画布层 · 全尺寸透明画布留住超出可见区的溢出像素(60%+ 设计有溢出)，避免旧做法在边界截断丢像素 → 完整可编辑。
  5. DMD 蒸馏 · 8 步 · Distribution Matching Distillation：最小化师生分布的 KL，50 步→8 步（NFE↓）、6× 提速(FID 16.02→18.58)。同 Qwen-Image-2.0/Drifting 的少步化。
  6. 三任务 · text-to-layers(文生整摞) / image-to-layers(平图拆成层=分割+补全) / layers-to-layers(加层 or 改风格)。一个模型靠 masking 全包。
