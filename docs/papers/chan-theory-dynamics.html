背驰与区间套。

缠中说禅 · 教你炒股票 108 课 · 背驰 / 区间套 / 买卖点篇

形态学告诉你走势"长什么样"，动力学回答"还有没有劲、什么时候转"。背驰是力气衰竭的信号，区间套把它精确到一个点 —— 这是缠论真正区别于普通技术分析、也最唬人的部分。

缠论三篇 · ① 形态学（主篇） · ② 动力学（本篇） · ③ 操作篇

TL;DR

形态学（分型→笔→线段→中枢）讲结构，动力学讲转折。核心是背驰1：同向趋势里，离开第二个中枢的力度比离开第一个弱（标准结构 a+A+b+B+c2），价格还创新高但劲6小了。力度用 MACD 面积5 量。分趋势背驰3（≥2 中枢，大概率反转）和盘整背驰4（1 中枢，弱）。背驰是个区间，区间套7（大级别背驰里套小级别背驰）把它精确到一个点。一条定理钉死动力学与买卖点8，三类买卖点91011各有出身且级别嵌套转化12。

§ 01背驰 · 力度的比较

背驰1 就一句话：价格还在创新高/新低，但力气明显小了 —— 趋势快走完了。缠论把它结构化成 a+A+b+B+c2：A、B 是同向趋势里相邻的两个中枢，a/b/c 是连接和离开它们的段。当离开第二个中枢的 c 段力度 < 离开第一个中枢的 a 段力度，就是背驰。

§ 02用 MACD 量"力度"

"力度"不能只靠眼看。缠论用 MACD5 量：B 中枢会把 MACD 黄白线拉回 0 轴附近，然后比较 c 段和 a 段对应的红绿柱面积（上涨看红柱、下跌看绿柱）—— c 段面积明显小，就是力度衰竭。

§ 04区间套 · 精确到一个点

背驰是一个区间（一段里劲在变小），不是精确的点。区间套7 的招很巧：c 段内部本身至少含两个次级别中枢，于是可以在 c 段里再套一个 a+A+b+B+c 做次级别的背驰判断；次级别里还能再套…… 大级别背驰的区间 ∩ 小级别也背驰的那一刻 = 精确买卖点。

§ 05定理 · 背驰 ↔ 买卖点

缠论用一条定理把动力学和买卖点焊死：

任何级别的背驰8，必然制造某级别的买卖点；任何级别的买卖点，必然源自某级别的背驰。

硬核, 但仍有主观空间

背驰 + 区间套是缠论最唬人的部分，给了普通技术分析没有的精确感。但 MACD 面积怎么比、c 段从哪算、套到哪个级别停 仍靠判断；背驰钝化（劲小了还继续走）是著名的坑。当成"趋势可能衰竭的预警 + 定位工具"，不是必然反转信号。完整争议见 形态学主篇。

引用 / 术语表

  1. 背驰 · 价格还在创新高/新低，但力度明显比上一段弱 → 趋势将尽。是缠论动力学的核心，跟普通技术分析的"背离"思路一致但有严格的结构和级别定义。
  2. 标准结构 a+A+b+B+c · 背驰的判断框架。A、B 是同向趋势里相邻的两个中枢，a/b/c 是连接与离开它们的段。比较"离开第二个中枢的 c 段"和"离开第一个中枢的 a 段"的力度。
  3. 趋势背驰 · 发生在有≥2 个同向中枢的趋势末端，c 段走完 MACD 面积小于 a 段 → 标准背驰，大概率反转，制造第一类买卖点。可靠性高。
  4. 盘整背驰 · 发生在只有 1 个中枢的盘整里，c 段不破中枢、MACD 面积变小 → 之后有回拉。级别小、可靠性低于趋势背驰。
  5. MACD 面积 / 黄白线 · 缠论量化"力度"的工具。中枢会把 MACD 黄白线拉回 0 轴附近；比较两段的红绿柱面积（涨看红、跌看绿）+ 黄白线回拉高度，面积明显小 = 力度衰竭。
  6. 力度 · 一段走势"劲有多大"，体现为斜率/速度，量化为 MACD 面积。背驰的本质就是"价更极端但力度更小"。
  7. 区间套 · 把背驰从"区间"精确到"点"的方法。c 段内含次级别中枢，可在 c 段里再做次级别背驰判断，层层嵌套；大级别背驰区间 ∩ 小级别也背驰的那一刻 = 精确买卖点。
  8. 背驰-买卖点定理 · 缠论把动力学和买卖点焊死的定理：任何级别的背驰必然制造某级别的买卖点；任何级别的买卖点必然源自某级别的背驰。
  9. 第一类买点 · 下跌趋势背驰后的反转最低点，由背驰直接造出来。风险最大、最便宜。
  10. 第二类买点 · 一买后第一次次级别回调结束、不破新低的点，本质 = 次级别回调走完。
  11. 第三类买点 · 中枢被向上突破后、回试不重新跌回中枢的点，确认中枢级别升级、趋势延续。最稳但已涨一段。
  12. 级别嵌套转化 · 买卖点在级别之间套娃：某级别的第三类买点，往往就是更大级别的第一/二类买点。小级别的终点是大级别的起点。
