FIELD NOTES · 循环 transformer · 视觉生成

ELT ·
写个 for 循环，不要复制粘贴。

Goyal · Kusupati 等 · Elastic Looped Transformers · 视觉生成 · arXiv 2604.09168 · 2026-04

生成模型靠堆很多不同的层换深度，参数也跟着堆上去。ELT 的赌注：把同一个 block 循环套 L 圈 —— 在深度上共享权重，参数只看 block、算力随圈数走，还能一次训练出一族深度，推理时按预算挑快或挑好。

TL;DR

ELT 把 looped transformer1 带进视觉生成（图像 + 视频）。不堆 D 个不同层，而是把 N 个 unique 层组成的 block 循环套 L 圈2：有效深度 = N×L，但参数只看 N、算力随 L 涨（权重共享3）。一次训练出一族深度 → Elastic / Any-Time 推理4，推理时挑 L 在质量-算力前沿上滑。训练靠 Intra-Loop Self Distillation (ILSD)5：teacher 跑满圈、student 随机中途退，加蒸馏逼每一档都好用。底座是 DiT7（在 SD VAE latent 上）和 MaskGIT/MAGVIT。结果：同等推理算力下 4× 参数缩减，ImageNet 256 FID 2.0、UCF-101 FVD 72.8、TPU v6e 上 3.5× 吞吐。

§ 01循环 · N 层 block 套 L 圈

标准 transformer 是 D 个各不相同的层叠起来，参数随深度涨。ELT 留一个 N 层的 block，把它循环套 L 圈——有效深度 = N×L，参数只存这 N 层。等于把"手写 100 个各不相同的函数"改成"写一个 block、for 循环跑 100 圈"。注意省的是参数（代码体积）不是算力——循环体每圈照跑，FLOPs 跟 100 个层一样多。

这跟 卷积的权重共享是同一招，只是共享的轴从"空间位置"换成了"深度 / 迭代"。详见 looped-transformer。

§ 02Elastic · 一次训练，一族深度

循环把 L 变成一个推理时的旋钮，于是一个权重文件天然带多档算力：手机跑 2 圈、服务器跑 10 圈，沿质量-算力的 Pareto 前沿滑——不用为每个预算单独训一个模型。这就是 Elastic / Any-Time4，跟 Matryoshka6 同一条弹性思路，只是旋钮做在循环圈数上。

§ 03ILSD · 怎么让每一档都好用

难点不在推理，在训练：怎么保证"中途退出"的浅档也像样，而不是只有满配能用？ELT 的 Intra-Loop Self Distillation5 在一步训练里同时练 teacher 和 student：

把 ELT 当图像生成想最顺：循环 = 在同一张画布上反复精修。每个训练步随机抽一个中途圈数 L_int，用同一个头读出 teacher（满 L_max → F_max，精修到底）和 student（L_int 退出 → F_int，粗版），三个 loss 全灌进同一套 Θ：真值(F_max) 锚住最好档、λ·真值(F_int) 让粗版自己也能用、蒸馏(F_int → stop-grad F_max) 把粗版拉向精修版。蒸馏 + stop-grad 是灵魂——让循环成为一条精修轨迹（多套一圈就更接近 F_max、保证"越多越好"），且只让 student 往 teacher 靠、不让弱的把强的拖down。净效果：共享 block 被逼成一个"单调精修器"，任意一圈停下都是张连贯（只是更粗）的输出。代价是训练更复杂。详见 elastic-inference。

ELT 是"权重共享"主题的又一个面：卷积在空间上共享、looped transformer 在深度上共享。两者都把"重复结构"从参数里抽出来，换成一个可复用的算子。它跟 ideogram-4 是视觉生成的两条省法：ideogram 省在"监督形态"（结构化 caption），ELT 省在"参数复用 + 弹性深度"。共同的味道——别把规律摊在参数里，把它做成结构。

引用 / 术语表

  1. looped transformer · 不堆很多不同层，而是把同一个 block 循环套用，在深度上共享权重。详见 looped-transformer。
  2. N×L · 有效深度 · N = 循环体里 unique 层数（定参数）；L = 循环圈数（定算力）。有效深度 = N×L。如 8N×4L = 等效 32 层、只存 8 层参数。L 在推理时可调。
  3. 权重共享 · weight sharing · 同一套参数被多次复用。卷积沿空间共享，looped transformer 沿深度/迭代共享 → 解耦"算多深"和"多少参数"。
  4. Elastic / Any-Time 推理 · 一次训练出一族深度，推理时按算力预算挑 L，在质量-算力 Pareto 前沿上滑，不必为每个预算单独训。详见 elastic-inference。
  5. ILSD · Intra-Loop Self Distillation · 训练技巧：teacher 跑满 L_max、student 随机在中间 L_int 退出，loss = 满配真值 + 中途真值(×λ) + 蒸馏(student 对齐 teacher，teacher stop-grad)，两路更新同一套 Θ → 逼每一档都好用。
  6. Matryoshka · 套娃 · 一套权重内嵌多个规模、推理时按需取一档的弹性思路（Kusupati 等）。ELT 把这个旋钮从"宽度/维度"挪到"循环圈数"。
  7. DiT · Diffusion Transformer · 把扩散去噪网络换成 transformer，在 latent（这里 SD v1.4 VAE）上做生成。ELT 把它的层换成循环 block。详见 diffusion-transformer、kl-vae。
  8. MaskGIT / MAGVIT · 掩码生成 · 另一类视觉生成：BERT 式 transformer 在 VQ token 上迭代地"填空"出图/视频。ELT 也在这套上做了循环版（图像 MaskGIT、视频 MAGVIT）。
  9. FID / FVD · 生成质量指标（越低越好）：FID 图像、FVD 视频，比真实分布与生成分布的距离。
