Ideogram 4.0 ·
把排版做进权重。

Ideogram Team · 9.3B 单流 DiT · 开源权重 · 2026-06-03

普通文生图吃一句话，版面和文字基本靠运气。Ideogram 4 的赌注：与其让模型从模糊语言里猜，不如把结构（位置、颜色、文字）显式写进 JSON，训练和推理都用它 —— 9.3B 的体量，把文本渲染和版面控制做到了开源第一。

TL;DR

Ideogram 第一个开源权重模型（9.3B）。骨架是单流 Diffusion Transformer1（34 层，文本+图像 token 一条序列、共享投影2，对比 SD3/FLUX 的双流 MMDiT3）。两个特别之处：文本编码器用 Qwen3-VL（VLM）4，且吃它 13 个中间层 hidden state 拼接5；模型只用结构化 JSON caption 训练8，显式给 bounding box9 和调色板10。训练目标 flow matching6，采样用 非对称 CFG7。frozen KL-VAE11 解码到像素。结果：9.3B 的文本渲染碾压 20B/32B/80B 的开源对手，设计 ELO 全开源第一、逼近闭源 GPT Image 2。

§ 02单流 DiT · 文本图像一条序列

骨架是 Diffusion Transformer1（把扩散去噪网络从 U-Net 换成 Transformer）。现代文生图都把文本和图像 token 放进 self-attention，但放法分两派：

每个 block 是标准料：self-attention 带 QK-RMSNorm12 和 3D Multimodal RoPE（把文本和图像 token 放进同一个位置系，RoPE 的多模态版）、SwiGLU MLP，时间步 t 通过 AdaLN 调制每层。详见 diffusion-transformer。

§ 03用 VLM 当文本编码器

第一个特别之处：文本编码器不是常见的 T5 或 CLIP，而是 Qwen3-VL-8B-Instruct4（一个视觉-语言模型，text-only 模式）。而且 DiT 不只取它最后一层，而是吃 13 个中间层的 hidden state 拼接5。

§ 04训练目标 · flow matching + 非对称 CFG

训练目标是 flow matching6：DiT 学一个速度场 v(z_t, t)，定义一条从纯噪声到干净 latent 的 ODE。采样时 Euler 把这条 ODE 积出来（12/20/48 步），噪声 schedule 随分辨率自适应。详见 flow-matching。

采样的引导用 非对称 CFG7。标准 CFG 同一步算两遍——带 prompt 的条件支、不带的无条件支——把差值放大逼模型听话。Ideogram 4 的"非对称"在于：无条件支把文本 token 整个丢掉（只跑图像 token），而不是换成 padding。单流架构天然能这么做。

v_cond   = DiT(z_t, t, text)       # 看着 prompt
v_uncond = DiT(z_t, t, ∅)          # 非对称: 文本整个 drop, 只跑图像 token
v = v_uncond + w · (v_cond − v_uncond)   # 放大差值

好处是两支能独立调，把"听话程度"和"画质"分开排期：V4_QUALITY_48 跑 45 步强引导（gw=7）+ 3 步 polish（gw=3）收细节，避免整体过饱和。详见 classifier-free-guidance。

§ 05杀手锏 · 结构化 JSON caption

这是 Ideogram 4 跟所有人最不一样的地方：模型只用结构化 JSON caption 训练8，不吃自然语言。每条 caption 穷举画面的每个元素 + style 块 + 可选 bbox + 调色板，训练和推理同一套格式，推理前还按 schema 校验。

三样普通 prompt 给不了的：调色板条件10（直接用 hex 控主色，每图 16 色）、bbox 布局9（[y_min,x_min,y_max,x_max] 归一化 0–1000，靠 MRoPE 落位）、文字元素（一字段放要渲染的字面文字、一字段放它长什么样 → 多行多字体图内文字）。普通用户写一句话也行——一个 "magic prompt" LLM 先把它扩成这套 JSON。详见 structured-caption-conditioning。

§ 06结果 · 9.3B 打赢一堆大模型

最能说明问题的是参数效率：9.3B 的 Ideogram 4，文本渲染（X-Omni OCR）打赢了 Qwen-Image (20B)、FLUX.2 dev (32B)、Hunyuan Image 3.0 (80B MoE)：

其它轴也硬：版面控制（7Bench mIoU 0.69，比所有闭源还高）、文本渲染 X-Omni OCR 0.97、空间推理 0.76、prompt 对齐 0.89。设计师盲评 ELO（4366 票）Ideogram 4 排第 2、全开源第 1，只输给闭源 GPT Image 2。

放进 wiki 这条线

9.3B 打赢 80B，再次印证 dMel / Whisper / fish-speech 那条主线：监督的形态 > 纯堆参数。Ideogram 4 的"形态"就是结构化 JSON caption —— "caption 把越多关系钉死，每对样本给的监督越扎实"。这跟 fish-speech 把对齐做进权重、dMel 把预处理塌缩是同一招的视觉版：把结构做进训练，而不是推理时拼 prompt。

引用 / 术语表

  1. Diffusion Transformer · DiT · 把扩散去噪网络从卷积 U-Net 换成 Transformer（Peebles & Xie 2023）。图像 latent 切成 token 进 self-attention，时间步用 AdaLN 调制。现代文生图（SD3/FLUX/Ideogram）的共同底座。详见 diffusion-transformer。
  2. 单流 · single-stream · 文本和图像 token 共享同一套投影（Q/K/V、MLP），每层就是一条 self-attention 序列。Ideogram 4（34 层）、Z-Image、Hunyuan 3 走这条。结构更简、参数更省。
  3. 双流 · MMDiT · 文本和图像各有一套独立投影，只在 attention 那一步互看，两条流并行（SD3、FLUX）。表达力强但参数多。单流是它的简化对照。
  4. Qwen3-VL 文本编码器 · Ideogram 4 的文本编码器是 Qwen3-VL-8B-Instruct（一个视觉-语言模型）的 text-only 模式，frozen 复用。用更懂语言的 VLM 替代常见的 T5/CLIP 文本编码器。
  5. 13 层 hidden state 拼接 · DiT 不只取文本编码器最后一层，而是把 Qwen3-VL 的 13 个中间层 hidden state 沿特征维拼接喂进去。对比 Qwen-Image 取单层、FLUX 无外部编码器 —— 取更多层 = 喂更丰富的语言表示。
  6. flow matching · Ideogram 4 的训练目标。DiT 学一个速度场 v(z_t,t)，定义从纯噪声到干净 latent 的 ODE；采样用 Euler 积分。比 diffusion 的 score matching 更直观。详见 flow-matching。
  7. 非对称 CFG · Classifier-Free Guidance 的变体。标准 CFG 无条件支把文本换 padding；Ideogram 4 把文本 token 整个丢掉，无条件支只跑图像 token。两支独立调，能把"听话程度"和"画质"分开排期（如 45 步强引导 + 3 步 polish）。详见 classifier-free-guidance。
  8. 结构化 JSON caption · Ideogram 4 的核心差异点：只用 JSON caption 训练，不吃自然语言。每条穷举画面每个元素 + style + bbox + 调色板，训练/推理同格式、推理前按 schema 校验。详见 structured-caption-conditioning。
  9. bounding-box 布局控制 · JSON 里每个元素可带 bbox，格式 [y_min,x_min,y_max,x_max] 归一化到 0–1000。模型靠共享的 MRoPE 位置系把元素放进框里 —— 布局控制不是外挂模块，是借位置编码天然实现的。7Bench mIoU 0.69 比闭源还高。
  10. 调色板条件 · JSON 里用 hex 色直接指定主色调（每图最多 16 色、每元素 5 色），不靠"蓝色调"这种形容词。让配色可控、可复现。
  11. KL VAE latent · 8× 压缩 · frozen 的 KL 自编码器（潜空间扩散的标配），把图像压成 8× 空间压缩的 latent（32×2²=128 通道），DiT 在 latent 空间去噪，最后 VAE 解码回 RGB 像素。
  12. MRoPE · QK-RMSNorm · 3D Multimodal RoPE 把文本和图像 token 放进同一个位置系（section 24/20/20），是 bbox 布局能落位的关键；QK-RMSNorm 对 attention 的 Q/K 做归一化稳训练。都是现代 Transformer 的标准件。
