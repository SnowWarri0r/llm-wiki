DINO ·
没有标签，自己教自己看图。

Caron et al. (Meta FAIR) · Emerging Properties in Self-Supervised ViT · arXiv 2104.14294 · 2021

怎么不用任何人工标签，让模型学会"看懂"图？DINO 的办法：一个学生网络去对齐一个老师网络的输出，而老师就是学生自己的滑动平均——自己教自己。训出来的 ViT 没人教分割，注意力却自动落在物体上。

TL;DR

DINO = self-DIstillation with NO labels1：学生和老师两个同构 ViT，**学生用梯度更新去对齐老师的输出分布**（交叉熵），**老师是学生的 EMA**2（动量平均、不回传梯度）。喂图用 multi-crop3：学生看全局+局部多个裁剪、老师只看全局 → 逼"局部也要预测出全局语义"。没有负样本、没有标签，靠 centering + sharpening4 两个力平衡来防坍缩。结果是涌现5：ViT 的 [CLS] 注意力无监督就落在物体上（≈ 免费分割），特征拿 k-NN 直接分类就很强。后续 DINOv2 把它 scale 成通用视觉基座——也是 perceptual loss 里 "DINO loss" 的来历。

§ 01自监督 · 没标签怎么学

监督学习要人标"这是猫"。自监督不用标签——自己从数据里造监督信号。DINO 造的信号是："同一张图的两个不同裁剪，模型对它们的输出应该一致"（都是这张图，理应编码出相近的东西）。难点是：怎么不退化成"全部输出同一个常数"（那也"一致"，但啥也没学到）。DINO 的全部精巧就在这——师生自蒸馏 + 防坍缩。

§ 02师生自蒸馏 · 老师是学生的影子

两个同构 ViT：student（梯度更新）和 teacher（不回传梯度）。同一张图给两边不同裁剪，让 student 的输出去对齐 teacher 的输出（softmax 成分布，算交叉熵）。关键问题：teacher 哪来？——teacher = student 的指数滑动平均（EMA），每步把 student 的权重以动量慢慢挪进 teacher。学生追一个"更稳、更慢的自己"。

注意：没有负样本（不像 contrastive learning 要推开不同图），也没有标签——只有"两个视角输出该一致"。也呼应 ELT 的 ILSD：都是"自蒸馏"，teacher 是更强/更稳的自己。

§ 03Multi-crop · 局部也要猜出全局

怎么让"一致"逼出有用的语义？靠裁剪的不对称：每张图切 2 个全局裁剪（大）+ 8 个局部裁剪（小）。student 看全部 10 个，teacher 只看 2 个全局。于是 student 拿一个局部小块（比如猫的耳朵）也得去对齐 teacher 看整只猫得到的输出——被逼着"从局部推出全局语义"。这是 DINO 学到物体级表征的关键。

§ 04防坍缩 · centering 和 sharpening 两个力

"两个视角输出一致"有个作弊解：所有图都输出同一个常数（也"一致"，但表征废了）= 坍缩。DINO 不用负样本防坍缩，而是都作用在 teacher 输出、进 softmax 之前，加两个相反方向的力：

teacher 分布 = softmax( ( g(x) − c ) / τ_t )
                         └ center ┘   └sharpen┘
  · sharpening = 那个很小的温度 τ_t(~0.04~0.07)。logits 除以小温度 → softmax 被拉尖。
  · centering  = 减去一个中心向量 c。c 是 teacher 输出的滑动均值(EMA)：
                 每步  c ← m·c + (1−m)·(这个 batch 里 teacher 输出的均值)
                 减掉这个均值 → 没有哪一维能长期占高位。

student 分布 = softmax( g(x) / τ_s )       # τ_s 较大(~0.1)，不减 c、不锐化
loss = − Σ  teacher 分布 · log(student 分布)   # 两个软分布的交叉熵

所以两个力都是 teacher softmax 的"前处理"——sharpening 是温度旋钮（除以小数→变尖），centering 是减掉平均水位（→变平）。一个推尖、一个推平，方向相反、互相拽住：

两个力方向相反、谁也压不死谁，分布就稳在"有结构但不坍缩"的中间带。少任何一个，训练都会出问题。无负样本、无标签，就靠这一对力撑住。

§ 05涌现 · 没人教却长出语义

最惊艳的是涌现属性：这么训出来的 ViT，[CLS] token 的自注意力图无监督就落在物体上（不同 head 还分到不同物体/部件），几乎是免费的前景分割；特征直接拿 k-NN 分类就达到强监督水平。没人教它"什么是物体"，语义自己长出来了。

能力DINO ViT（无监督）
[CLS] 注意力自动聚焦物体 → 近似分割掩码
k-NN 分类（ImageNet）不微调、不训分类头就很强
特征迁移检测/分割/检索下游通吃
放进 wiki 这条线

DINO 把"自蒸馏"这招用到了极致——而你已经见过它两次：ELT 的 ILSD（student 中途退、对齐满配 teacher）、还有这里（student 对齐 EMA teacher）。共同骨架是"让一个网络去追一个更稳/更强版本的自己"。DINO 的特征因为强在语义/结构，被 perceptual loss 借去当 "DINO loss"——HiDream-O1 拿它补像素扩散缺的长程语义连贯，正是看中这点。后续 DINOv2(2023) 把它 scale 成通用视觉特征基座。

引用 / 术语表

  1. 自监督 · self-supervised · 不用人工标签，从数据本身造监督信号（这里："同一图两个裁剪的输出该一致"）。对比监督学习要标"这是猫"。
  2. EMA teacher · 动量老师 · teacher 不直接训，而是 student 权重的指数滑动平均（每步动量挪一点）。teacher 更稳、变化更慢，给 student 一个稳定的追逐目标；teacher 侧 stop-grad。
  3. multi-crop · 多裁剪 · 每图切 2 个全局裁剪 + 8 个局部裁剪。student 看全部、teacher 只看全局 → 逼 student 从局部预测出全局语义（"局部到全局"）。
  4. centering / sharpening · 防坍缩的一对相反力：sharpening（teacher softmax 低温）把分布推尖防均匀坍缩；centering（减滑动均值）把分布推平防单维独大。平衡才稳。
  5. 涌现属性 · emerging properties · 无监督训练却自动出现的能力：[CLS] 注意力聚焦物体≈分割、k-NN 特征强分类。语义不是教出来的，是训出来的。
  6. DINOv2 · 2021 DINO 的放大版（2023），更大数据/模型，成为通用视觉特征基座，被各类下游和 perceptual loss 广泛借用。
