2026-07-20paperHDR · 先规划整段轨迹，再逐层生成每一步HDR 不让流式视频模型看到第一步就立刻押死整条路线，而是先生成一棵从粗到细的时间层级：上层用少量、仍带噪的 latent 表示可修改的整体方案，下层再把方案细化成连续画面；SHAP 只连接同层前一步、父节点和相邻父节点，把时间注意力从平方级降到线性级。 2026-07-20concept流式 AR vs 双向视频扩散双向扩散让整段视频在每个去噪 step 互相修改，推理强但代价高；流式 AR 从左到右提交，延迟低但早期错误无法撤回。 2026-07-20conceptConditional Flow Matching · 条件流匹配把难算的全局 velocity field 训练目标，换成"给定一对噪声和数据样本时该往哪走"的可监督回归。 2026-07-20conceptHierarchical Latent Denoising · 层级 latent 去噪先用少量粗时间节点决定整段轨迹，再用更多细节点逐层落实；粗层故意不完全去噪，避免太早把一种方案锁死。 2026-07-20conceptKV CacheTransformer 每层 attention 算过的 Key / Value 矩阵存下来，下一个 token 直接复用，不重算前面所有 token。 2026-07-20conceptSHAP · 稀疏层级注意力树中每个节点只读同层前一个节点、自己的父节点、父节点左右邻居和层首参考帧，用固定数量的连接传递长程规划。 2026-07-17paperLTX-2 · 让声音和画面在同一段去噪过程中彼此校正**视频负责空间和运动，音频负责时间和声场；两条流各自去噪，又在每一层沿共同时间轴双向对答案。** 2026-07-17conceptAdaptive LayerNorm · AdaLN / AdaLN-Zero · 自适应层归一化不用固定的归一化缩放/平移，而是用"条件"(时间步 t + 类别/文字 c)临时算出来——让条件去实时拧每一层的旋钮。 2026-07-17conceptAsymmetric Dual-Stream Transformer · 不对称双流 Transformer音频和视频各走一条专用流水线，层数对齐、宽度不同，再在每层交换信息。 2026-07-17conceptAudio VAE · 把波形压成适合生成模型处理的声音 latent先把密集波形变成时频图，再压成低帧率 latent；生成完成后由 decoder 和 vocoder 还原声音。 2026-07-17conceptAudiovisual Cross-Attention · 音视频双向交叉注意力视频拿自己的问题去查声音，声音也拿自己的问题去查视频，并用共同时间轴对齐事件。 2026-07-17conceptClassifier-Free Guidance · CFG让模型同一步算两遍 —— 一遍看着 prompt、一遍蒙着眼，再把"看着 prompt 多走的那部分"放大，逼它更听话。 2026-07-17conceptCross-Attention · 交叉注意力Decoder 用自己的 Query，去查 Encoder 输出的 Key/Value，让生成过程能看见输入序列。 2026-07-17conceptDiffusion Transformer · DiT把扩散模型里那个老 U-Net 换成 Transformer —— 图像切成 latent token，像处理一串 token 一样去噪。
