2026-06-29paperGRAPE · 最好的指令微调数据，是"合身"的那条同一条指令、好几个老师给的候选回答，别默认挑最强老师那条；让**目标模型自己**给每个候选打分（长度归一概率 = 困惑度），选它觉得**最自然/最合身**（困惑度最低）的那条来做 SFT。结果超过最强老师(405B)、超过 3× 大的数据集，还省算力。NeurIPS 2025 spotlight。 2026-06-29concept分布对齐的 SFT · 数据好不好，是相对你这个模型说的微调时别只挑"绝对最强"的回答，挑**贴合目标模型自己分布**的那条（它给高概率/低困惑度）——既学到任务，又少把模型从预训练分布上拽走。 2026-06-29concept困惑度 Perplexity · 模型对一段话有多"意外"把模型在一段文本上的平均交叉熵取指数，得到一个"等效分支数"：困惑度 = exp(平均每 token 的负对数概率)；越低 = 模型觉得这段话越自然、越在意料之中。 2026-06-28paperReLAT · 给潜在推理"闭环"：测试时用重建当保真检查潜在推理把中间思考从自然语言搬进不透明的潜在向量里图省事，但没人检查这些向量是否还忠于原问题（开环）。ReLAT 在答题前先做一圈 **问题→潜在思考→重建问题**：如果从潜在向量重建得回原问题，说明它没丢信息；用这个重建损失在测试时临时调一组 LoRA，再正式作答，答完即弃。 2026-06-28concept可微思考表示 · 别硬选一个 token，取嵌入的加权混合要让"选哪个 token"这一步能反传梯度，就不硬选（argmax 不可导），而是对词表 logits 做 softmax 得分布 α，用它加权各 token 的嵌入算一个**期望嵌入**——既留在嵌入空间里，又对下游损失可微。 2026-06-28concept潜在推理 · 把"想"压进连续向量，不摊成语言不把推理的中间步写成自然语言（CoT），而是产出一串连续潜在向量当"思考"，被下游当条件反复用——省 token、但不透明，还可能悄悄丢了原问题的约束。 2026-06-28concept分层图像生成 · 产出一摞可编辑图层，不是一张拍平图生成的不是糊在一个平面上的像素图，而是一摞带透明通道（RGBA）的图层（画布+背景+K 个前景），每层能单独挪/改/重组——图像版的"逐元素编辑"。 2026-06-28concept重建即保真 · 能还原回输入，就说明中间表示没丢信息要检查一个中间表示是否忠于输入，不必有标签——只问"能不能从它**重建回原输入**"：还原得回，信息就在；还原不回，关键信息已经丢了。 2026-06-28concept选择性 token masking · 用"哪些给定/哪些待生成"当任务开关同一个扩散模型，靠"哪些 token 从干净 latent（给定、不预测）初始化、哪些从噪声（待生成）初始化"这个开关，把生成/分解/编辑统一成一个模型——换 mask = 换任务，权重不动。 2026-06-28concept测试时训练 TTT · 每道题临时调一下参数，答完即弃推理时对**当前这一个输入**用一个无监督辅助损失临时更新参数（通常只动 LoRA），调好后作答，然后把更新**复位**——下一题重来。 2026-06-26conceptDINOv3 多样性奖励 · 给"组内整齐划一"扣钱，防风格塌缩一条 prompt 生成一组图，用 DINOv3 把每张压成向量，算组内两两的平均距离当"多样性分"，加进奖励——逼模型每次给的几张互不雷同。 2026-06-26conceptFAISS / 近似最近邻 · 用一点精度换十亿级的速度精确最近邻要拿查询跟全部向量逐个比，十亿级太慢；FAISS 用"先分桶只翻几桶"(IVF)+"把向量压成几字节"(PQ)两招，换来毫秒级近似检索。 2026-06-26concept层级 k-means 策展 · 把"逐张审查"压成"逐簇审查"十亿张图没法一张张看；先用 k-means 把它们按视觉语义聚成簇、再递归切子簇，每个簇心附近挑几张代表图让 VLM 审一眼，按簇整批保留/丢弃/补采。 2026-06-26conceptPageRank 实体覆盖 · 用链接结构挑出"必须见过的概念"拿维基百科的链接图跑 PageRank，给每个条目算个"重要度"，取排名前 90% 当"必须覆盖的概念"（约 500 万个），再回头查训练集有没有图文对覆盖到它们。
