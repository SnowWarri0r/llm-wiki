2026-06-29paperGRAPE · 最好的指令微调数据，是"合身"的那条同一条指令、好几个老师给的候选回答，别默认挑最强老师那条；让**目标模型自己**给每个候选打分（长度归一概率 = 困惑度），选它觉得**最自然/最合身**（困惑度最低）的那条来做 SFT。结果超过最强老师(405B)、超过 3× 大的数据集，还省算力。NeurIPS 2025 spotlight。 2026-06-29paperOmniEraser · 用视频帧白送的 ground-truth 教模型"连影子一起抹掉"对象消除的真痛点不是"把物体抠掉",而是**连它的影子/反光一起干净抹除、且别凭空补出个新东西**。OmniEraser 两手都解决:数据上用**视频帧白送真 ground-truth**([[video-frame-paired-supervision]])——物体在场的帧当输入、物体离开后的帧当目标,影子在后一帧里真的没了;模型上用**双条件输入**([[object-background-guidance]])把"要删的物体"也喂进去,让 [[flux-1]] DiT 明确知道哪块该清空,不再瞎补。 2026-06-29concept背景建模(MOG) · 固定机位下,谁老不变就是背景,谁突然变就是前景固定机位拍的视频里,背景的每个像素长期稳定、前景(走动的物体)经过时才突变;给每个像素建一个"它平时是什么颜色"的概率模型(高斯混合),某帧某像素偏离太多就判为前景——这样不用人工就能从视频自动挑出"有物体 / 没物体"的帧。 2026-06-29concept分布对齐的 SFT · 数据好不好，是相对你这个模型说的微调时别只挑"绝对最强"的回答，挑**贴合目标模型自己分布**的那条（它给高概率/低困惑度）——既学到任务，又少把模型从预训练分布上拽走。 2026-06-29concept双条件输入 · 不光告诉模型"周围长啥样",还告诉它"要删的是这个"普通 inpaint 只把**背景上下文**喂给扩散模型,模型不知道挖空的洞里"本该是空的",于是照周围猜着补出个新物体;OmniEraser 把**要删的物体本身**也作为一路条件一起喂进去,模型就明确知道"这块是要清掉的东西、那块是要保留的背景",不再瞎补。 2026-06-29concept对象效果消除 · 抹的不只是物体,还有它在世界上留下的痕迹真正的"删除一个物体"要连它的**影子、反光、倒影**一起抹掉,并把那块还原成合理的空背景;只抠掉物体本体、留下一摊孤零零的影子,看着比不删还假。 2026-06-29concepton-policy vs off-policy · 学自己刚做的，还是学别人/旧的on-policy = 拿**当前策略自己刚生成**的数据学（信号对得准、稳，但得不断现采）；off-policy = 拿**别的策略**（旧的自己/别的模型/人类）的数据学（能复用、便宜，但分布错位、要纠偏）。 2026-06-29concept困惑度 Perplexity · 模型对一段话有多"意外"把模型在一段文本上的平均交叉熵取指数，得到一个"等效分支数"：困惑度 = exp(平均每 token 的负对数概率)；越低 = 模型觉得这段话越自然、越在意料之中。 2026-06-29concept视频帧白送的成对监督 · 物体自己会走,等于免费拍了 before/after要教模型"把物体连影子一起抹掉",最难的是拿到 ground-truth(抹干净后的真实画面);视频里物体本来就会移动,**有物体的那一帧当输入、物体离开后的那一帧当目标**,影子/反光在后一帧里真的没了——监督信号白送,不用手工修图。 2026-06-28paperReLAT · 给潜在推理"闭环"：测试时用重建当保真检查潜在推理把中间思考从自然语言搬进不透明的潜在向量里图省事，但没人检查这些向量是否还忠于原问题（开环）。ReLAT 在答题前先做一圈 **问题→潜在思考→重建问题**：如果从潜在向量重建得回原问题，说明它没丢信息；用这个重建损失在测试时临时调一组 LoRA，再正式作答，答完即弃。 2026-06-28concept可微思考表示 · 别硬选一个 token，取嵌入的加权混合要让"选哪个 token"这一步能反传梯度，就不硬选（argmax 不可导），而是对词表 logits 做 softmax 得分布 α，用它加权各 token 的嵌入算一个**期望嵌入**——既留在嵌入空间里，又对下游损失可微。 2026-06-28concept潜在推理 · 把"想"压进连续向量，不摊成语言不把推理的中间步写成自然语言（CoT），而是产出一串连续潜在向量当"思考"，被下游当条件反复用——省 token、但不透明，还可能悄悄丢了原问题的约束。 2026-06-28concept分层图像生成 · 产出一摞可编辑图层，不是一张拍平图生成的不是糊在一个平面上的像素图，而是一摞带透明通道（RGBA）的图层（画布+背景+K 个前景），每层能单独挪/改/重组——图像版的"逐元素编辑"。 2026-06-28concept重建即保真 · 能还原回输入，就说明中间表示没丢信息要检查一个中间表示是否忠于输入，不必有标签——只问"能不能从它**重建回原输入**"：还原得回，信息就在；还原不回，关键信息已经丢了。
